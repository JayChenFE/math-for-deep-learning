# 附录 D：全书知识交汇索引图

> 本书 35 章不是孤岛——几乎每一章都依赖前置章节的概念，同时为后续章节埋下伏笔。本附录用**五张"交汇地图"**展示全书的知识网络，帮你理解"为什么学这个"以及"这个以后用在哪"。

---

## D.1 四大"知识融合点"

全书有四个关键节点——多根之前的知识线在此处汇合，形成强大的"复合工具"。

### 融合点 1：第 12 章 — 梯度向量的首次融合

```
第 3 章(向量)  ─┐
第 10 章(导数) ─┼─→ 第 12 章：梯度 ∇f = [∂f/∂x₁, ..., ∂f/∂xₙ]
第 11 章(偏导) ─┘         │
                           ├─→ 第 13 章(链式法则)
                           ├─→ 第 24 章(优化算法)
                           └─→ 第 28 章(训练循环)
```

**总结**：梯度 = 导数(Ch10)的向量(Ch03)化 + 偏导(Ch11)的打包。理解它=理解整个 DL 训练的基础。

### 融合点 2：第 17 章 — 协方差矩阵的特征分解

```
第 8 章(特征值) ─┐
第 15 章(分布)   ─┼─→ 第 17 章：协方差矩阵 → 特征分解 → PCA
第 17 章(协方差) ─┘         │
                             └─→ 第 9 章(SVD 实现 PCA)
                             └─→ 第 25 章(权重初始化—保持方差)
                             └─→ 第 34 章(LoRA—低秩≈保留大特征值方向)
```

**总结**：特征分解(Ch08)不是抽象数学——通过协方差矩阵(Ch17)找到了 AI 的核心应用：PCA 降维。

### 融合点 3：第 19 章 — 信息论与损失函数

```
第 15 章(MLE)     ─┐
第 16 章(贝叶斯)  ─┼─→ 第 19 章：交叉熵 = MLE 的对数版本
第 12 章(梯度)    ─┘         │
                             └─→ 第 27 章(逻辑回归的损失)
                             └─→ 第 28 章(两层网络的损失)
                             └─→ 第 31 章(训练循环中的 loss)
                             └─→ 第 33 章(解码时的采样策略)
```

**总结**：你见过的所有分类损失函数 = 交叉熵(Ch19)的变体。理解它=理解"模型为什么这样被惩罚"。

### 融合点 4：第 29~35 章 — Transformer 的终极融合

```
第 4 章(残差连接) ───→ 第 30 章(Pre-Norm + x + Sublayer(x))
第 5 章(点积)    ────→ 第 29 章(Q·K 注意力得分)
第 6 章(矩阵乘法) ──→ 第 29 章(QKV 投影、W_O 输出投影)
第 9 章(SVD)     ────→ 第 34 章(LoRA = 低秩更新)
第 15 章(采样)   ────→ 第 33 章(Temperature/Top-p)
第 21 章(LN)     ────→ 第 30 章(Pre-Norm)
第 22 章(Softmax)───→ 第 29 章(稳定版 softmax)、第 35 章(Logit Bias)
第 24 章(AdamW)  ────→ 第 31 章(optimizer.step())、第 34 章(微调)
第 32 章(KV Cache)──→ 第 33 章(解码时复用)
```

**总结**：Transformer 不是"新技术"，是你前面 28 章学的每一个数学概念的**总装配线**。

## D.4 纵向纵向交叉切面

### "相似度"的数学进化链

```
Ch05 点积(两向量) → Ch05 余弦相似度(归一化点积)
    → Ch06 矩阵乘法(批量点积)
    → Ch29 Q@Kᵀ(注意力=软件化动态相似度)
    → Ch34 LoRA(B@A=低秩相似度近似)
```

### "损失函数"的来源链

```
Ch15 MLE(概率视角) → Ch19 熵+KL+交叉熵(信息论视角)
    → Ch19 MLE⇔交叉熵(等价证明)
    → Ch27 二分类交叉熵
    → Ch28 多分类交叉熵
    → Ch31 训练循环中的 loss.backward()
```

### "优化"的升级链

```
Ch02 割线斜率(手动) → Ch12 梯度下降(一阶迭代)
    → Ch24 Momentum(SGD+惯性)
    → Ch24 Adam(自适应学习率)
    → Ch24 AdamW(解耦Weight Decay)
    → Ch31 training loop forward→loss→backward→step
```

### "归一化"的进化链

```
Ch20 浮点精度(为啥需要归一化) → Ch21 BatchNorm(跨batch)
    → Ch21 LayerNorm(跨特征) → Ch21 RMSNorm(去均值)
    → Ch30 Pre-Norm(x+LN(x)+Sublayer)
```
