## B.1 对数（Logarithm）

**定义**：若 `bʸ = x`，则 `log_b(x) = y`。对数回答"b 的几次方等于 x？"

| 底数 | 记号 | 别名 | 在本书中的应用 |
|------|------|------|---------------|
| 2 | `log₂(x)` | 二进制对数 | 信息量单位 bit（第 19 章） |
| e ≈ 2.718 | `ln(x)` | 自然对数 | 交叉熵损失、MLE 推导（第 15/19 章） |
| 10 | `log₁₀(x)` | 常用对数 | 偶尔用于画图 |

**关键性质**：

| 性质 | 公式 | 为什么重要 |
|------|------|-----------|
| 乘法变加法 | `log(a · b) = log(a) + log(b)` | MLE 取 log 后乘法变加法—优化更容易（第 15.5 节） |
| 幂变乘 | `log(aⁿ) = n · log(a)` | 指数族分布的 log 似然（第 15 章） |
| 倒数变号 | `log(1/a) = −log(a)` | 交叉熵 = −log(p_correct) |
| 换底公式 | `log_b(x) = ln(x) / ln(b)` | 信息论中 bits ↔ nats 转换（第 19 章） |
| 自然对数的导数 | `d/dx[ln(x)] = 1/x` | 交叉熵反向传播推导 |

💡 **直觉**：对数把"乘"变成"加"，把"大数"压缩成"中数"——`log₁₀(1000000) = 6`。深度学习用它来稳定数值计算和简化优化。

## B.3 求和记号 Σ

`Σ_{i=1}^{n} xᵢ` = `x₁ + x₂ + ... + xₙ`

- 均值：`x̄ = (1/n) Σ xᵢ`
- 点积：`a·b = Σ aᵢ bᵢ`
- 期望：`E[X] = Σ x · P(X=x)`
- 熵：`H = −Σ pᵢ log(pᵢ)`

## B.4 求根公式（二次方程）

对于 `ax² + bx + c = 0`（a ≠ 0）：

$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

- Δ = b² − 4ac > 0 → 两个不同实根
- Δ = 0 → 一个重根
- Δ < 0 → 两个共轭复根

本书不直接使用求根公式，但它帮助你理解"解析解"的概念——和线性回归闭式解 `w = (XᵀX)⁻¹Xᵀy`（第 7 章）是同一哲学：有时候可以一步到位，不需要迭代。

## B.7 向量的基本概念

- 向量有**方向**和**大小**（长度/范数）
- `||v||₂ = √(v₁² + v₂² + ... + v_n²)`（L2 范数 / 欧几里得长度）
- 两个向量的夹角 θ：`cosθ = (a·b) / (||a||·||b||)`
- 垂直（正交）：`a·b = 0`
- 同向：`a·b > 0`，反向：`a·b < 0`
