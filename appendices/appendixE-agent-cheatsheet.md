## E.1 快速诊断索引

| 遇到的问题 | 根因章节 | 解决方案 |
|-----------|---------|---------|
| 模型输出 NaN | Ch22(softmax 溢出) | `x - x.max()` 稳定版本；或降低 logits 量级 |
| 训练 loss 突然爆炸→NaN | Ch23(梯度爆炸) | `clip_grad_norm_(model.parameters(), 1.0)` |
| 梯度更新后参数不变 | Ch20(浮点精度) | 梯度 < 1e-7 在 fp32 下更新失效 → 用混合精度或增大 lr |
| 推理时显存随序列长度暴涨 | Ch32(KV Cache) | 检查 `use_cache=True`；考虑 PagedAttention(vLLM) |
| 多轮对话显存持续增长 | Ch14(计算图) | 每轮对话后用 `.detach()` 切断历史梯度 |
| 生成文本不断重复 | Ch33(解码策略) | 用 Top-p(p=0.9)+T(0.7-0.9)+Repetition Penalty 替代 Greedy |
| JSON 输出解析失败率高 | Ch35(结构化生成) | 降低 T 至 0.1-0.3；加 Logit Bias；用 Guidance/Outlines |
| 微调后模型丧失通用能力 | Ch34(灾难性遗忘) | 用 LoRA(r=8) 替代全量微调；降低学习率 |
| 训练初期 loss 震荡剧烈 | Ch24(Warmup) | 前 2000-4000 步线性 warmup |
| 小 batch 梯度估计太噪声 | Ch24(梯度累积) | `gradient_accumulation_steps=4` 模拟大 batch |
| 深层网络中浅层梯度为零 | Ch23/Ch25(消失/初始化) | 用 Kaiming 初始化 + Pre-Norm 残差连接 |
| 权重初始化后输出方差漂移 | Ch25(初始化) | ReLU → Kaiming；Tanh/Sigmoid → Xavier |
| 全零初始化所有神经元输出相同 | Ch25(对称性) | 必须随机初始化——全零 = 对称性灾难 |
