### 8.1　定义——矩阵作用下"方向不变"的向量

📐 **定义　特征向量与特征值（Eigenvector & Eigenvalue）**：对于方阵 **A**，若存在非零向量 **v** 和标量 λ 满足 `**A**·**v** = λ·**v**`，则 **v** 是 **A** 的一个特征向量，λ 是对应的特征值。

通俗理解：矩阵乘法通常会旋转向量。但**特征向量是那类特殊向量——被矩阵乘了之后，方向完全不变，只被缩放了 λ 倍**。

💻 **代码　验证特征方程**

```python
import numpy as np

A = np.array([[4, 1],
              [2, 3]])

# 计算特征值和特征向量
eigvals, eigvecs = np.linalg.eig(A)
# eigvecs[:, i] 是对应于 eigvals[i] 的特征向量

print("特征值:")
for i, lam in enumerate(eigvals):
    print(f"  λ{i+1} = {lam:.3f}")

# 验证：A @ v = λ @ v
for i in range(2):
    v = eigvecs[:, i]
    lam = eigvals[i]
    Av = A @ v
    lam_v = lam * v
    print(f"\n验证特征向量 {i+1}:")
    print(f"  A @ v  = {Av}")
    print(f"  λ * v  = {lam_v}")
    print(f"  匹配: {np.allclose(Av, lam_v)}")
```

### 8.2　几何可视化——在平面上看"不旋转的方向"

💻 **代码　特征向量方向可视化**

```python
import numpy as np
import matplotlib.pyplot as plt

A = np.array([[2, 0.5],
              [0.5, 1]])
eigvals, eigvecs = np.linalg.eig(A)

fig, ax = plt.subplots(figsize=(6, 6))
ax.set_xlim(-5, 5); ax.set_ylim(-5, 5)
ax.axhline(0, color='gray', linewidth=0.5)
ax.axvline(0, color='gray', linewidth=0.5)

# 画单位圆上的点变换前后
theta = np.linspace(0, 2*np.pi, 100)
circle = np.column_stack([np.cos(theta), np.sin(theta)])  # (100, 2)
transformed = circle @ A.T  # (100, 2)

ax.plot(circle[:, 0], circle[:, 1], 'b-', alpha=0.3, label='单位圆')
ax.plot(transformed[:, 0], transformed[:, 1], 'r-', alpha=0.5, label='变换后（椭圆）')

# 画特征向量方向（放大便于观察）
colors = ['green', 'orange']
for i in range(2):
    v = eigvecs[:, i]
    lam = eigvals[i]
    ax.arrow(0, 0, v[0]*lam*2, v[1]*lam*2, head_width=0.15,
             color=colors[i], label=f'λ{i+1}={lam:.2f}')

ax.set_aspect('equal'); ax.legend()
ax.set_title('特征向量：单位圆→椭圆 的拉伸轴')
plt.show()
```

单位圆在矩阵 **A** 作用下变成一个椭圆。椭圆的**长轴和短轴方向就是特征向量**，对应特征值的大小决定了在这个方向上被拉伸了多少。这就是 PCA 降维的几何原型——保留大特征值方向（主成分），丢弃小特征值方向。

🔗 **AI 连接**：PCA（主成分分析，第 17 章）的核心就是对协方差矩阵做特征分解，取最大 k 个特征值对应的特征向量作为投影方向。特征值和特征向量也是理解 SVD（第 9 章）和 LoRA 低秩微调的前置概念。

### 8.3　特征值的含义——信息的"浓度"

对于对称矩阵（如协方差矩阵），所有特征值都是实数。将它们从大到小排序：

- **大特征值** → 对应的方向数据方差大 → "信息丰富"
- **小特征值** → 数据在该方向几乎不变 → 可丢弃（降维）

💻 **代码　特征值衰减与信息占比**

```python
import numpy as np

# 随机对称矩阵（模拟协方差矩阵）
np.random.seed(42)
B = np.random.randn(5, 5)
A = B.T @ B  # 对称正定
eigvals = np.sort(np.linalg.eigvalsh(A))[::-1]  # 从大到小排序
# 注：eigvalsh 用于对称/Hermitian 矩阵，返回实数特征值

print("特征值 (从大到小):")
for i, lam in enumerate(eigvals):
    ratio = lam / eigvals.sum() * 100
    cum = eigvals[:i+1].sum() / eigvals.sum() * 100
    print(f"  λ{i+1} = {lam:.3f}  占比 {ratio:5.1f}%  累计 {cum:5.1f}%")
print(f"\n前 2 个特征值解释了 {eigvals[:2].sum()/eigvals.sum()*100:.0f}% 的信息")
# 通常前几个特征值就占了总能量的绝大部分
```

这就是 PCA 的核心直觉：前 k 个特征值对应的特征向量捕获了数据中大部分的变异（方差），用它们做投影可以在保留信息的前提下大幅降维。
