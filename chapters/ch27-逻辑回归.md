## 第27章　逻辑回归 —— 概率视角的分类器

> 本章目标：实现 sigmoid + 交叉熵的完整逻辑回归，手动推导梯度并更新参数，可视化决策边界。
> 前置知识：第 10 章（sigmoid 导数）、第 19 章（交叉熵）、第 26 章（线性回归）

---

💻 **代码　逻辑回归完整实现**

```python
import numpy as np
import matplotlib.pyplot as plt

def sigmoid(z):
    z = np.clip(z, -500, 500)  # 防溢出
    return 1 / (1 + np.exp(-z))

# 生成二分类数据
np.random.seed(42)
N = 200
X_pos = np.random.randn(N//2, 2) + np.array([2, 2])
X_neg = np.random.randn(N//2, 2) + np.array([-2, -2])
X = np.vstack([X_pos, X_neg])
y = np.hstack([np.ones(N//2), np.zeros(N//2)])

# 梯度下降训练
X_aug = np.column_stack([X, np.ones(N)])
w = np.zeros(3); lr = 0.1
for epoch in range(500):
    z = X_aug @ w
    p = sigmoid(z)
    grad = X_aug.T @ (p - y) / N
    w -= lr * grad
    if epoch % 200 == 0:
        loss = -np.mean(y * np.log(p + 1e-8) + (1-y) * np.log(1-p + 1e-8))
        print(f"epoch {epoch}: loss={loss:.4f}")

# 决策边界可视化
x1 = np.linspace(-5, 5, 100); x2 = -(w[0]*x1 + w[2]) / w[1]
plt.scatter(X_pos[:,0], X_pos[:,1], c='blue', alpha=0.5, label='class 1')
plt.scatter(X_neg[:,0], X_neg[:,1], c='red', alpha=0.5, label='class 0')
plt.plot(x1, x2, 'k-', label='decision boundary')
plt.legend(); plt.show()
```
