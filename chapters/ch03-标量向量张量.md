## 第3章　标量、向量与张量 —— AI的数据容器

> 本章目标：理解 AI 中数据的四种维度——标量（0维）、向量（1维）、矩阵（2维）、张量（3维+），掌握 `shape` 和 `axis` 这两个贯穿全书的核心概念。读完本章，你将能正确解释任何张量运算的形状变化。
> 前置知识：第 1 章（Python 基础、NumPy 数组）

---

### 3.1　标量（Scalar）——0 维数据

📐 **定义　标量（Scalar）**：一个单纯的数值，没有方向，没有维度。在 NumPy 中，标量是一个 0 维数组——`shape = ()`。

在深度学习中，最著名的标量就是**损失值（loss）**。训练神经网络时，整个前向传播 + 反向传播的终极目标就是把 `loss` 这个标量降得越小越好。

💻 **代码　认识标量**

```python
import numpy as np

# 标量就是一个数
loss = 3.14159
print(f"loss 值: {loss}")

# NumPy 中的 0 维数组
loss_np = np.array(3.14159)
print(f"ndim: {loss_np.ndim}")    # 0
print(f"shape: {loss_np.shape}")  # ()
print(f"用 .item() 取回 Python 数值: {loss_np.item()}")
# 在 PyTorch 中，loss.item() 是同样的操作
```

🔗 **AI 连接**：`loss.item()` 是 PyTorch 训练循环中最常见的操作之一——把 GPU 上的 0 维张量转成 Python float 来打印或记录。

---

### 3.2　向量（Vector）——1 维数据

📐 **定义　向量（Vector）**：一组有序的数值，可看作高维空间中的一个箭头。在 NumPy 中是一维数组——`shape = (n,)`。

💻 **代码　创建和操作向量**

```python
import numpy as np

# 用户特征向量：[年龄, 身高, 月收入, 活跃度]
user = np.array([25, 175, 8000, 0.85])
print(f"user shape: {user.shape}")  # (4,)
print(f"user[0] (年龄): {user[0]}")

# 等差数列
x = np.linspace(0, 1, 5)  # 0 到 1 均匀取 5 个点
print(f"linspace: {x}")    # [0.  0.25 0.5  0.75 1. ]
```

> ⚠️ NumPy 一维数组的 shape 是 `(n,)` 而不是 `(n, 1)`。本书遵守 CLAUDE.md 符号约定：向量用粗体小写 **x**，代码中为 `x = np.array([...])`（shape 为 `(d,)` 的一维张量）。

---

### 3.3　矩阵（Matrix）——2 维数据

📐 **定义　矩阵（Matrix）**：一个二维数表，有行和列。在 NumPy 中是二维数组——`shape = (m, n)`。

在深度学习中，矩阵最常见的形态是**批量输入**：`X.shape = (batch_size, d_in)`。

💻 **代码　创建和检查矩阵**

```python
import numpy as np

# 模拟：32 个样本，每个 784 维（如 MNIST 的 28×28 展平）
X = np.random.randn(32, 784)
print(f"X.shape: {X.shape}")   # (32, 784)
print(f"样本数 (batch): {X.shape[0]}")
print(f"特征数 (d_in):  {X.shape[1]}")

# 取第一个样本
first_sample = X[0]   # shape: (784,)
print(f"第一个样本 shape: {first_sample.shape}")

# 取所有样本的第一个特征（第一列）
first_feature = X[:, 0]  # shape: (32,)
print(f"所有样本的第一个特征 shape: {first_feature.shape}")
```

🔗 **AI 连接**：`(batch, d_in)` 这个 2D shape 贯穿整个 PyTorch 代码库——`nn.Linear` 的输入、Embedding 的输出、分类头的输入都是这个形状。

---

### 3.4　张量（Tensor）——3 维及以上

📐 **定义　张量（Tensor）**：标量、向量、矩阵的泛化。3 阶张量是一个三维数组。本书用粗体书法体 **𝒳** 表示三阶及以上张量，或直接用 shape 标注。

深度学习中最典型的 3 阶张量：**Transformer 输入** `(batch, seq_len, d_model)`——这个 shape 将在第 29 章反复出现。

💻 **代码　创建和理解 3 阶张量**

```python
import numpy as np

# Transformer 一个 batch 的输入: (batch=4, seq_len=10, d_model=512)
X_transformer = np.random.randn(4, 10, 512)
print(f"Transformer 输入 shape: {X_transformer.shape}")
print(f"  维度 0 (batch):    {X_transformer.shape[0]} 个句子")
print(f"  维度 1 (seq_len):  {X_transformer.shape[1]} 个 token")
print(f"  维度 2 (d_model):  {X_transformer.shape[2]} 维嵌入")

# 取第 0 句第 3 个 token 的全部 512 维嵌入
token_embedding = X_transformer[0, 3, :]
print(f"第 0 句第 3 个 token 的嵌入 shape: {token_embedding.shape}")  # (512,)

# 图片批次: (8, 3, 224, 224) —— 4 阶张量
images = np.random.randn(8, 3, 224, 224)
print(f"\n图片批次 shape: {images.shape}")
print(f"  这是一个 {images.ndim} 阶张量")
```

### 3.5　轴（Axis）——维度操作的导航仪

📐 **定义　轴（Axis）**：张量每个维度的索引号。`axis=0` 是第一个维度（通常是 batch），`axis=1` 是第二个。理解 axis 是掌握 `sum`、`mean`、`softmax`、`concatenate` 等所有维度操作的前提。

💻 **代码　axis 实战：np.sum**

```python
import numpy as np

A = np.array([[1, 2, 3],
              [4, 5, 6]])
print(f"A.shape: {A.shape}\nA:\n{A}\n")

# axis=0：沿行方向压缩 → 每列求和
print(f"sum(axis=0): {A.sum(axis=0)}  shape={A.sum(axis=0).shape}")
# [1+4, 2+5, 3+6] = [5, 7, 9]  → shape (3,)

# axis=1：沿列方向压缩 → 每行求和
print(f"sum(axis=1): {A.sum(axis=1)}  shape={A.sum(axis=1).shape}")
# [1+2+3, 4+5+6] = [6, 15]     → shape (2,)

# 对 3 阶张量：axis 轴被"消掉"
T = np.random.randn(4, 10, 512)
print(f"\nT.shape = (4, 10, 512)")
print(f"T.sum(axis=0).shape = {T.sum(axis=0).shape}")  # (10, 512)
print(f"T.sum(axis=1).shape = {T.sum(axis=1).shape}")  # (4, 512)
print(f"T.sum(axis=2).shape = {T.sum(axis=2).shape}")  # (4, 10)
```

关键规律：**沿哪个 axis 操作，结果的 shape 就抹掉哪个维度**。适用于 `sum`、`mean`、`max`、`softmax(dim=...)`。

🔗 **AI 连接**：`softmax(dim=-1)` 在最后一维上做归一化——这行代码在第 22 章和第 29 章会出现无数次。
