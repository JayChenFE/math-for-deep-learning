## 第26章　线性回归 —— 全流程闭环

> 本章目标：用闭式解和梯度下降两种方法求解线性回归，对比优劣，画出拟合结果。
> 前置知识：第 7 章（闭式解）、第 12 章（梯度下降）

---

💻 **代码　全流程：合成数据 → 闭式解 → 梯度下降 → 对比**

```python
import numpy as np
import matplotlib.pyplot as plt

np.random.seed(42)
N = 100; X = np.random.randn(N, 1)
true_w, true_b = 2.0, 1.0
y = X.ravel() * true_w + true_b + np.random.randn(N) * 0.5

# 闭式解
X_aug = np.column_stack([X, np.ones(N)])
w_closed = np.linalg.inv(X_aug.T @ X_aug) @ X_aug.T @ y

# 梯度下降
w_gd = np.array([0.0, 0.0]); lr = 0.01
for _ in range(1000):
    y_pred = X_aug @ w_gd
    grad = -2/N * X_aug.T @ (y - y_pred)
    w_gd -= lr * grad

print(f"闭式解: w={w_closed[0]:.3f}, b={w_closed[1]:.3f}")
print(f"梯度下降: w={w_gd[0]:.3f}, b={w_gd[1]:.3f}")
print(f"真实值: w={true_w}, b={true_b}")

# 画图
plt.scatter(X.ravel(), y, alpha=0.5, label='data')
x_line = np.linspace(-3, 3, 100)
plt.plot(x_line, w_closed[0]*x_line + w_closed[1], 'r-', label='fit')
plt.xlabel('x'); plt.ylabel('y'); plt.legend(); plt.show()
```
