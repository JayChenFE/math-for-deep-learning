## 第28章　两层神经网络 —— NumPy手写反向传播

> 本章目标：用纯 NumPy 实现输入(784)→隐藏(256,ReLU)→输出(10,Softmax)的两层网络，手写前向/反向传播，跑通 MNIST 达到 ~92% 准确率。
> 前置知识：第 6 章（全连接层）、第 10 章（ReLU/Softmax 导数）、第 13 章（链式法则）、第 19 章（交叉熵）

---

💻 **代码　两层神经网络——纯 NumPy**

```python
import numpy as np

def relu(x): return np.maximum(0, x)
def relu_grad(x): return (x > 0).astype(float)
def softmax_stable(x):
    x = x - x.max(axis=1, keepdims=True)
    e = np.exp(x); return e / e.sum(axis=1, keepdims=True)

# 网络参数
np.random.seed(42)
d_in, d_hidden, d_out = 784, 256, 10
W1 = np.random.randn(d_hidden, d_in) * np.sqrt(2./d_in)
b1 = np.zeros(d_hidden)
W2 = np.random.randn(d_out, d_hidden) * np.sqrt(2./d_hidden)
b2 = np.zeros(d_out)

# 模拟一个 batch
N = 64
X = np.random.randn(N, d_in)
y_true = np.random.randint(0, d_out, N)

# 前向传播
h_pre = X @ W1.T + b1          # (64, 256)
h = relu(h_pre)                 # (64, 256)
logits = h @ W2.T + b2          # (64, 10)
probs = softmax_stable(logits)  # (64, 10)

# 交叉熵损失
loss = -np.log(probs[np.arange(N), y_true] + 1e-8).mean()
print(f"Loss: {loss:.4f}")

# 反向传播
d_logits = probs.copy()
d_logits[np.arange(N), y_true] -= 1
d_logits /= N

dW2 = d_logits.T @ h             # (10, 256)
db2 = d_logits.sum(axis=0)       # (10,)
dh = d_logits @ W2               # (64, 256)
dh_pre = dh * relu_grad(h_pre)   # ReLU backward
dW1 = dh_pre.T @ X               # (256, 784)
db1 = dh_pre.sum(axis=0)         # (256,)

# 更新
lr = 0.01
W1 -= lr * dW1; b1 -= lr * db1
W2 -= lr * dW2; b2 -= lr * db2
print(f"Gradient norms: dW1={np.linalg.norm(dW1):.3f}, dW2={np.linalg.norm(dW2):.3f}")
print("反向传播完成 ✓")
```
