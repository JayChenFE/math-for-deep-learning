## 第19章　信息论 —— 损失函数的终极来源

> 本章目标：理解熵、KL 散度、交叉熵，推导"最大化似然 = 最小化交叉熵"，掌握困惑度的直觉含义。
> 前置知识：第 15 章（概率分布）、第 16 章（条件概率）

---

### 19.1~19.2　信息量与熵

📐 **信息量**：I(p) = −log₂(p)。低概率事件发生带来更多"惊讶"=更多信息。
📐 **熵（Entropy）**：H(P) = −Σ pᵢ·log(pᵢ)。系统的平均不确定性。均匀分布熵最大。

### 19.3~19.4　KL 散度与交叉熵

📐 **KL 散度**：D_KL(P‖Q) = Σ pᵢ·log(pᵢ/qᵢ)。用分布 Q 来编码真实分布 P 时"多花的比特数"。
📐 **交叉熵**：H(P,Q) = −Σ pᵢ·log(qᵢ) = H(P) + D_KL(P‖Q)。

💻 **代码　熵与交叉熵**

```python
import numpy as np

def entropy(p):
    p = np.array(p); p = p[p > 0]
    return -np.sum(p * np.log2(p))

def cross_entropy(p, q):
    p, q = np.array(p), np.array(q)
    return -np.sum(p * np.log2(q + 1e-12))

p_true = [0.7, 0.3]
print(f"均匀硬币 H: {entropy([0.5,0.5]):.3f} bit")
print(f"偏置硬币 H: {entropy(p_true):.3f} bit")
print(f"交叉熵(p,q): {cross_entropy(p_true, [0.6,0.4]):.3f}")
print(f"交叉熵(p,p): {cross_entropy(p_true, p_true):.3f} = H(p)")
```

### 19.5　交叉熵 = 极大似然 + 取负对数

📐 最大似然估计等价于最小化交叉熵——因为 `log P(D|θ) = Σ log q(y|x)`，即 `−cross_entropy`。Softmax + 交叉熵是分类问题的标准损失函数。

### 19.6　困惑度（Perplexity）

📐 **Perplexity = exp(cross_entropy)**。直觉：perp=10 意味着"模型在每个位置从 10 个等概率 token 中猜"。perp 越低越好。

💻 **代码　困惑度计算**

```python
def perplexity(log_probs):
    return np.exp(-np.mean(log_probs))

log_probs_good = np.log([0.8, 0.7, 0.9, 0.6, 0.85])
log_probs_bad = np.log([0.3, 0.2, 0.1, 0.4, 0.25])
print(f"Good model perplexity: {perplexity(log_probs_good):.1f}")
print(f"Bad model perplexity:  {perplexity(log_probs_bad):.1f}")
```

🔗 **AI 连接**：`nn.CrossEntropyLoss()` = log_softmax + NLLLoss。第 27 章逻辑回归和第 30 章 GPT 训练都用它。
