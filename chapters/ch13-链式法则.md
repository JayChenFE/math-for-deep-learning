## 第13章　链式法则 —— 嵌套函数的"拆包"

> 本章目标：掌握链式法则 `dh/dx = (dh/du)·(du/dx)`，理解为什么"外层导数乘内层导数"能处理任意深度嵌套——这是反向传播能工作的数学根基。
> 前置知识：第 10 章（导数公式）、第 12 章（梯度）

---

### 13.1　嵌套函数的拆解

对于 h(x) = sin(x²)，设 u = x², h = sin(u)。链式法则：`dh/dx = dh/du · du/dx = cos(u) · 2x = cos(x²)·2x`。

📐 **定义　链式法则（Chain Rule）**：`(f∘g)'(x) = f'(g(x)) · g'(x)`。多层嵌套逐层相乘。

---

### 13.2　数值验证

💻 **代码　数值微分 vs 链式法则**

```python
import numpy as np

def numerical_deriv(f, x, h=1e-5):
    return (f(x+h) - f(x-h)) / (2*h)

# h(x) = sin(x²)
h = lambda x: np.sin(x**2)
u = lambda x: x**2       # 内层
du_dx = lambda x: 2*x    # 内层导数
dh_du = lambda u: np.cos(u)  # 外层导数

x0 = 2.0
chain = dh_du(u(x0)) * du_dx(x0)  # cos(4) * 4
numeric = numerical_deriv(h, x0)
print(f"链式法则: {chain:.6f}")
print(f"数值微分: {numeric:.6f}")
print(f"匹配: {abs(chain-numeric)<1e-5} ✓")
```

---

### 13.3　三层嵌套

h(x) = ln(cos(x³))：拆为 u=x³, v=cos(u), h=ln(v)。链式：`dh/dx = (1/v)·(−sin(u))·(3x²)`。

💻 **代码　自动展开任意嵌套**

```python
def chain_demo(x0=1.0):
    # h(x) = ln(cos(x³))
    u = x0**3; v = np.cos(u); h = np.log(v)
    # 逐层导数
    dh_dv = 1/v; dv_du = -np.sin(u); du_dx = 3*x0**2
    chain = dh_dv * dv_du * du_dx
    # 数值验证
    h_func = lambda x: np.log(np.cos(x**3))
    numeric = numerical_deriv(h_func, x0)
    print(f"链式法则: {chain:.6f}  数值: {numeric:.6f}  匹配: {abs(chain-numeric)<1e-4}")

chain_demo(1.0)
```

### 13.4　神经网络 = 链式法则的超长链条

📐 **核心洞察**：L = loss(model(x)) 对第一层权重 W₁ 的梯度 = `∂L/∂hₙ · ∂hₙ/∂hₙ₋₁ · ... · ∂h₁/∂W₁`。这就是反向传播——从最后一层开始逐层往前乘。
