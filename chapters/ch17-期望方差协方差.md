### 17.1~17.2　期望与方差

📐 **期望（Expectation）**：`E[X] = Σ x·P(x)` 或 `np.mean(X)`，加权平均。
📐 **方差（Variance）**：`Var(X) = E[(X−E[X])²]`，衡量数据分散程度。

### 17.3　协方差与协方差矩阵

📐 **协方差（Covariance）**：`Cov(X,Y) = E[(X−E[X])(Y−E[Y])]`。正值=同向变化，负值=反向变化，0=不相关。

📐 **协方差矩阵**：Σ[i,j] = Cov(Xᵢ, Xⱼ)。对角线是方差，非对角线是协方差。

💻 **代码　协方差矩阵与热力图**

```python
import numpy as np
import matplotlib.pyplot as plt

np.random.seed(42)
X = np.random.randn(100, 4)
X[:, 1] = X[:, 0] * 0.8 + np.random.randn(100) * 0.3  # feat1 与 feat0 正相关
X[:, 2] = -X[:, 0] * 0.6 + np.random.randn(100) * 0.3 # feat2 与 feat0 负相关

cov_mat = np.cov(X.T)
print("协方差矩阵:\n", cov_mat.round(2))

plt.imshow(cov_mat, cmap='RdBu_r', vmin=-2, vmax=2)
plt.colorbar(label='covariance'); plt.title('Covariance Matrix Heatmap')
plt.xticks(range(4), [f'feat{i}' for i in range(4)])
plt.yticks(range(4), [f'feat{i}' for i in range(4)]); plt.show()
```

### 17.4　PCA 降维——协方差矩阵的特征分解

📐 **PCA 步骤**：(1) 数据中心化 (2) 计算协方差矩阵 (3) 特征分解 (4) 取前 k 大特征值对应的特征向量做投影。这正是第 8 章特征向量在真实数据上的应用。

💻 **代码　NumPy 实现 PCA**

```python
X_centered = X - X.mean(axis=0)
cov = X_centered.T @ X_centered / (len(X) - 1)
eigvals, eigvecs = np.linalg.eigh(cov)
top2 = eigvecs[:, -2:]  # 最大两个特征值对应的向量
X_pca = X_centered @ top2  # 降维到 2D
print(f"原始 shape: {X.shape}  →  PCA后: {X_pca.shape}")
print(f"解释方差比: {eigvals[-2:][::-1] / eigvals.sum()}")
```

🔗 **AI 连接**：PCA 是数据可视化和降噪的经典工具。虽然深度学习通常让网络自己学特征，但 PCA 的"协方差→特征分解"思维链直接通向了 SVD（第 9 章）和 LoRA。
