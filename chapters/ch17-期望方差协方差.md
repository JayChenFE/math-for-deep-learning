## 第17章　期望、方差与协方差矩阵 —— 第二次知识融合

> 本章目标：掌握描述数据分布的三个核心统计量——期望（数据中心）、方差（数据分散度）、协方差（两变量联动关系）。将协方差矩阵与第 8 章的特征分解融合：协方差矩阵的特征向量 = 数据变化的主方向（PCA 降维的数学核心）。最终用 NumPy 从零实现 PCA，把高维数据投影到二维平面。

### 17.1　期望 —— 数据的"重心"

你有一组数据 [3, 7, 2, 9, 4]。它们的"典型值"是多少？把所有值加起来除以个数——这就是**期望**（均值）。数学记号 E[X]，代码 `np.mean(X)`。

📐 **定义　期望（Expected Value / Mean）**：E[X] = Σ xᵢ·P(X=xᵢ)（离散）或 ∫ x·f(x)dx（连续）。样本均值 x̄ = (1/n) Σ xᵢ 是期望的无偏估计。

💻 **代码　期望的计算与"无偏性"直觉**

```python
import numpy as np
np.random.seed(42)

# 真实分布：均值为 5 的正态分布
true_mean = 5.0
sample_sizes = [5, 20, 100, 1000]

for n in sample_sizes:
    sample = np.random.randn(n) + true_mean
    sample_mean = np.mean(sample)
    print(f"n={n:4d}: x̄ = {sample_mean:.4f}, 误差 = {abs(sample_mean - true_mean):.4f}")
```

> **关键洞察**：n=5 时 x̄ 可能在 4.2~5.8 之间剧烈波动，n=1000 时 x̄ 稳定在 4.95~5.05。样本越多，样本均值越接近真实期望——这就是大数定律（第 18 章）的核心直觉。

🔗 **AI 连接**：BatchNorm（第 21 章）用 mini-batch 均值做中心化；损失函数 `loss.mean()` 是所有样本损失的平均；注意力机制中 softmax 的输出 E[**V**_i] = Σ p_i·**v**_i 是 Value 的加权期望。

### 17.2　方差与标准差 —— 数据的"离散度"

两组数据可以有相同的均值，但分布完全不同：[2, 3, 4, 5, 6] 和 [−8, 1, 4, 7, 16]——均值都是 4，但后者散得多。方差衡量这种**分散程度**。

📐 **定义　方差（Variance）与标准差（Standard Deviation）**：Var[X] = E[(X−μ)²] = E[X²] − (E[X])²。标准差 σ = √Var[X]，和原始数据同单位，更直观。

- σ 小 → 数据紧凑，集中在均值附近
- σ 大 → 数据分散，样本差异大

在 AI 中，方差过大是过拟合的信号（模型对训练数据的微小波动过度敏感），方差为 0 意味着所有数据完全相同（如全零初始化导致的对称性陷阱，第 25 章）。

💻 **代码　期望、方差、标准差：同样的均值，不同的方差**

```python
import numpy as np

np.random.seed(42)
# 两组均值相同但方差不同的数据
tight = np.random.normal(5, 1, 1000)    # σ=1, 集中在 5 附近
wide = np.random.normal(5, 5, 1000)     # σ=5, 散布很开

for name, data in [("tight (σ=1)", tight), ("wide (σ=5)", wide)]:
    print(f"{name}: mean={data.mean():.3f}, std={data.std():.3f}, var={data.var():.3f}")

# 方差 = E[X²] - (E[X])²  验证
var_formula = np.mean(tight**2) - np.mean(tight)**2
print(f"\nVar 等价公式验证: np.var={tight.var():.3f}, E[X²]-(E[X])²={var_formula:.3f} ✓")
```

> **关键洞察**：均值告诉你"中心在哪"，方差告诉你"数据散得多开"。两者共同描述了分布的基本形状。BN（第 21 章）的操作正是：先减均值（中心化），再除标准差（缩放至单位方差）。

🔗 **AI 连接**：BatchNorm（第 21 章）的公式 `(x−μ)/σ` 就是期望和方差的直接应用；权重初始化（第 25 章）的目标是让每层输出的方差保持一致。

### 17.3　协方差 —— 两个变量的"共变关系"

身高和体重——高的人通常更重。这就是**协方差**（Covariance）：衡量两个变量是"同变"（一个增大另一个也增大）还是"反变"（一个增大另一个减小）。

📐 **定义　协方差（Covariance）**：Cov(X, Y) = E[(X−μₓ)(Y−μᵧ)]。结果 > 0 → 正相关（同增同减），< 0 → 负相关（此消彼长），= 0 → 线性无关。

协方差的大小受数据单位影响（身高的 cm 换成 m，协方差缩小 100 倍）。为此引入**相关系数**（Correlation）：ρ = Cov(X,Y)/(σₓ·σᵧ)，标准化到 [−1, 1]，去除了单位影响。

💻 **代码　生成相关变量 + 计算协方差/相关系数**

```python
import numpy as np
import matplotlib.pyplot as plt

np.random.seed(42)
n = 500

# 构造正相关数据：身高 → 体重（加噪声）
height = np.random.normal(170, 10, n)          # 身高 (cm)
weight = 0.8 * height + np.random.normal(0, 8, n)  # 体重 = 0.8×身高 + 噪声

# 构造负相关数据：温度 → 取暖费
temp = np.random.normal(15, 8, n)
heating = -3 * temp + np.random.normal(0, 10, n)

fig, axes = plt.subplots(1, 2, figsize=(12, 5))
for ax, x, y, title, cov_sign in [
    (axes[0], height, weight, '身高 vs 体重 (正相关)', 'positive'),
    (axes[1], temp, heating, '温度 vs 取暖费 (负相关)', 'negative')]:
    ax.scatter(x, y, alpha=0.4, s=8)
    cov = np.cov(x, y)[0, 1]
    corr = np.corrcoef(x, y)[0, 1]
    ax.set_xlabel('x'); ax.set_ylabel('y')
    ax.set_title(f'{title}\n协方差={cov:.1f}, 相关系数={corr:.3f}')

plt.tight_layout(); plt.show()
print("协方差 > 0 = 同向变化; 协方差 < 0 = 反向变化; 相关系数 ∈ [-1,1]")
```

> **关键洞察**：协方差矩阵 = 方差 + 协方差的统一表格。对角线是各变量的方差，非对角线是两两之间的协方差。这个矩阵是 PCA 的输入——**特征分解协方差矩阵 = 找到数据变化最大的方向。**

### 17.4　协方差矩阵 —— 多特征相关性热力图

将 17.3 节的协方差扩展到 n 个特征：**协方差矩阵 Σ** 是一个 n×n 的对称矩阵，Σ[i,i] = Var(特征ᵢ)，Σ[i,j] = Cov(特征ᵢ, 特征ⱼ)。对角线是各特征自身的方差，非对角线是两两之间的协方差。

📐 **定义　协方差矩阵（Covariance Matrix）**：Σ = (1/n) (**X** − x̄)ᵀ (**X** − x̄)。对称正半定矩阵——它的特征向量方向 = 数据变化的主方向（PCA 的数学根基）。

💻 **代码　协方差矩阵热力图**

```python
import numpy as np
np.random.seed(42)

# 模拟 4 个相关特征
n = 500
x1 = np.random.randn(n)
x2 = 0.8 * x1 + np.random.randn(n) * 0.5      # 与 x1 强正相关
x3 = -0.6 * x1 + np.random.randn(n) * 0.7     # 与 x1 负相关
x4 = np.random.randn(n) * 1.5                  # 独立特征
X = np.column_stack([x1, x2, x3, x4])

# 协方差矩阵
cov_matrix = np.cov(X, rowvar=False)
print("协方差矩阵:\n", cov_matrix.round(3))
print(f"\nx1-x2 协方差 = {cov_matrix[0,1]:.3f} (>0 → 正相关)")
print(f"x1-x3 协方差 = {cov_matrix[0,2]:.3f} (<0 → 负相关)")
print(f"x1-x4 协方差 = {cov_matrix[0,3]:.3f} (≈0 → 无关)")
```

> **关键洞察**：协方差矩阵的非对角线是"特征对之间的线性关系指纹"。正数=同向变化，负数=反向变化，零=线性无关。PCA（17.5 节）找到的特征向量 = 这个矩阵的"主方向"。

🔗 **AI 连接**：PCA（17.5 节）的输入就是协方差矩阵。协方差矩阵的特征分解 → 最大特征值对应的特征向量 → 数据方差最大的方向 → 降维投影轴。第 8 章学的特征分解在这里找到了它最重要的 AI 应用。

---

### 17.5　AI 连接：协方差矩阵的特征分解 → PCA 降维

现在融合第 8 章的特征分解：**对协方差矩阵做特征分解，特征值最大的特征向量方向 = 数据方差最大的方向 = "数据变化的主方向"。** PCA（主成分分析）就是把数据投影到前 k 个最大特征值对应的特征向量上——在保留最大方差的同时将数据从 n 维降到 k 维。

📐 **定义　PCA（Principal Component Analysis）**：①数据中心化（减均值）；②计算协方差矩阵；③特征分解协方差矩阵；④取前 k 个最大特征值对应的特征向量组成投影矩阵；⑤数据 × 投影矩阵 → 降维结果。

💻 **代码　从零实现 PCA：用 Iris 数据集做 4D→2D 降维**

```python
import numpy as np
import matplotlib.pyplot as plt

# 生成模拟的 3 类数据（模拟 Iris：4 个特征）
np.random.seed(42)
n_per_class = 50
# 三类数据，各有不同的均值和协方差结构
class0 = np.random.randn(n_per_class, 4) * 0.5 + np.array([2, 3, 1, 0])
class1 = np.random.randn(n_per_class, 4) * 0.6 + np.array([-1, -1, 0, 2])
class2 = np.random.randn(n_per_class, 4) * 0.4 + np.array([3, -2, 2, -1])
X = np.vstack([class0, class1, class2])
y = np.array([0]*n_per_class + [1]*n_per_class + [2]*n_per_class)

# ===== PCA 从零实现 =====
# 1. 中心化
X_centered = X - X.mean(axis=0)

# 2. 协方差矩阵 (4×4)
cov_matrix = np.cov(X_centered, rowvar=False)

# 3. 特征分解
# 协方差矩阵是对称正半定的 → 特征值保证为实数。
# .real 是安全措施：np.linalg.eig 对非对称矩阵可能返回复数（CLAUDE.md 已知坑 #3）
eigvals, eigvecs = np.linalg.eig(cov_matrix)
eigvals = eigvals.real
eigvecs = eigvecs.real

# 4. 按特征值降序取前 k=2 个特征向量
idx = np.argsort(eigvals)[::-1]
W = eigvecs[:, idx[:2]]  # 投影矩阵 (4×2)

# 5. 投影
X_pca = X_centered @ W  # (150, 4) @ (4, 2) → (150, 2)

# 能量占比
total_var = eigvals.sum()
print(f"特征值: {eigvals[idx].round(3)}")
for i in range(4):
    print(f"  PC{i+1}: {eigvals[idx[i]]:.3f} ({eigvals[idx[i]]/total_var*100:.1f}%)")
print(f"前2维保留方差: {eigvals[idx[:2]].sum()/total_var*100:.1f}%")

# 可视化
fig, axes = plt.subplots(1, 2, figsize=(12, 5))
colors = ['red', 'green', 'blue']
for c in range(3):
    mask = y == c
    axes[0].scatter(X_pca[mask, 0], X_pca[mask, 1], c=colors[c], alpha=0.6, s=15, label=f'Class {c}')
axes[0].set_xlabel('PC1'); axes[0].set_ylabel('PC2')
axes[0].set_title('PCA: 4D → 2D (前2个主成分)'); axes[0].legend()

# 协方差矩阵热力图
im = axes[1].imshow(cov_matrix, cmap='RdBu_r', aspect='auto')
axes[1].set_title('协方差矩阵 (4×4)'); axes[1].set_xticks(range(4)); axes[1].set_yticks(range(4))
plt.colorbar(im, ax=axes[1]); plt.tight_layout(); plt.show()
```

> **关键洞察**：PCA 不是魔法——它就是第 8 章特征分解的直接应用。协方差矩阵告诉你"特征之间怎么一起变"，特征分解告诉你"这些变化的独立方向及其重要性排名"，投影就是"只保留最重要的几个方向"。整个流程：**协方差矩阵 → 特征分解 → 取前 k 个 → 投影 = 降维。**

🔗 **AI 连接**：PCA 用于数据预处理（降维提速 + 去噪）、特征提取、可视化高维数据。第 9 章 SVD 可以用更稳定的方式实现 PCA（对中心化数据直接 SVD，不需要显式计算协方差矩阵）。现代深度学习中，自编码器（Autoencoder）和 VAE 提供了比 PCA 更强大的非线性降维能力。
