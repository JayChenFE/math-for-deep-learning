> 本章目标：理解离散/连续概率分布、正态/均匀/Bernoulli分布、极大似然估计的直觉，掌握 temperature/top-k/top-p 四种采样策略。
> 前置知识：第 1 章（NumPy）、第 3 章（向量）

### 15.1~15.4　四大核心分布

📐 **均匀分布（Uniform）**：区间内每个值等概率。权重初始化最常用 `U(-a, a)`。
📐 **正态分布（Normal/Gaussian）**：钟形曲线，μ 控制中心，σ 控制宽度。神经网络参数初始化默认。
📐 **Bernoulli 分布**：二元结果（0/1），如点击率预测。

💻 **代码　生成分布样本**

```python
import numpy as np
import matplotlib.pyplot as plt

np.random.seed(42)
uniform = np.random.uniform(-1, 1, 10000)
normal = np.random.randn(10000)
bernoulli = np.random.binomial(1, 0.3, 10000)

fig, axes = plt.subplots(1, 3, figsize=(14, 3))
axes[0].hist(uniform, bins=50, alpha=0.7); axes[0].set_title('Uniform(-1,1)')
axes[1].hist(normal, bins=50, alpha=0.7); axes[1].set_title('Normal(0,1)')
axes[2].hist(bernoulli, bins=2, alpha=0.7); axes[2].set_title('Bernoulli(p=0.3)')
plt.tight_layout(); plt.show()
```

### 15.5　极大似然估计（MLE）直觉

📐 **MLE**：给定观测数据，找最可能产生这些数据的参数。对正态分布，μ 的 MLE 就是样本均值，σ² 的 MLE 就是样本方差。

💻 **代码　MLE 估计正态分布均值**

```python
np.random.seed(0)
data = np.random.randn(1000) * 2 + 5  # 真实 μ=5, σ=2
mu_mle = data.mean()
sigma_mle = data.std()
print(f"MLE: μ={mu_mle:.3f} (真实=5), σ={sigma_mle:.3f} (真实=2)")
```

### 15.6　采样策略：从概率分布到具体选择

📐 **四种策略**：
- **Greedy**：永远选概率最高的 → 文本重复绕圈
- **Temperature**：`softmax(x/T)`，T→0 退化为 greedy，T→∞ 变为均匀
- **Top-k**：只从概率前 k 高的 token 中采样
- **Top-p（核采样）**：从概率累计到 p 的最小 token 集合中采样（动态 k）

💻 **代码　四种采样策略对比**

```python
def softmax(x, T=1.0):
    x = np.array(x, dtype=float)
    e = np.exp((x - x.max()) / T)
    return e / e.sum()

logits = [2.0, 1.0, 0.5, 0.1, -1.0]
print("Greedy:", np.argmax(logits))
print("T=0.5:", softmax(logits, 0.5).round(3))
print("T=2.0:", softmax(logits, 2.0).round(3))

# Top-k sampling
def top_k_sample(logits, k):
    idx = np.argsort(logits)[-k:]
    probs = softmax(np.array(logits)[idx])
    return idx[np.random.choice(k, p=probs)]

np.random.seed(0)
samples = [top_k_sample(logits, 3) for _ in range(100)]
print(f"Top-3 samples distribution: {np.bincount(samples, minlength=5)}")
```
