### 梯度爆炸模拟

深度网络中梯度逐层相乘，若每层导数 >1，则深层梯度指数级增长。

📐 **梯度裁剪（Gradient Clipping）**：`if ‖g‖ > threshold: g = g * threshold / ‖g‖`。PyTorch 中 `torch.nn.utils.clip_grad_norm_(model.parameters(), max_norm)`。

💻 **代码　梯度爆炸与裁剪**

```python
import numpy as np

# 模拟梯度爆炸
g = np.array([1.0])
for _ in range(20):
    g = g * 2.0  # 每层翻倍
print(f"爆炸后的梯度: {g[0]:.0f}")

# 梯度裁剪
np.random.seed(42); grad = np.random.randn(100) * 100
norm = np.linalg.norm(grad)
threshold = 1.0
if norm > threshold:
    grad = grad * threshold / norm
print(f"裁剪前范数: {norm:.1f} → 裁剪后范数: {np.linalg.norm(grad):.3f}")
```

🔗 **AI 连接**：训练 RNN/LSTM 和 Transformer 时，梯度裁剪是标配——防止一个异常 batch 让参数飞到 NaN。
