## 第11章　偏导数 —— 按住一个，动另一个

> 本章目标：理解多元函数的偏导数（固定其他变量只对一个变量求导），用 `np.gradient` 计算二维网格上的偏导数，通过等高线图建立偏导→梯度的几何直觉。
> 前置知识：第 10 章（导数）、第 6 章（矩阵）、第 3 章（shape）

---

### 11.1　二元函数的偏导数

📐 **定义　偏导数（Partial Derivative）**：对于 f(x,y)，∂f/∂x 表示固定 y、只对 x 求导；∂f/∂y 表示固定 x、只对 y 求导。记号为 ∂ 而非 d。

对于 f(x,y) = x² + y²：∂f/∂x = 2x（把 y² 当常数），∂f/∂y = 2y（把 x² 当常数）。

💻 **代码　数值偏导数**

```python
import numpy as np

def f(x, y):
    return x**2 + y**2

def partial_x(f, x, y, h=1e-5):
    return (f(x+h, y) - f(x-h, y)) / (2*h)

def partial_y(f, x, y, h=1e-5):
    return (f(x, y+h) - f(x, y-h)) / (2*h)

x0, y0 = 3.0, 4.0
print(f"∂f/∂x at (3,4): 数值={partial_x(f,x0,y0):.1f}  解析={2*x0:.1f}")
print(f"∂f/∂y at (3,4): 数值={partial_y(f,x0,y0):.1f}  解析={2*y0:.1f}")
```

### 11.2　用 np.gradient 批量求偏导

`np.gradient` 对整个二维网格一次性求所有点的偏导数，无需循环。

💻 **代码　二维网格上的梯度**

```python
import numpy as np

x = np.linspace(-3, 3, 50); y = np.linspace(-3, 3, 50)
X, Y = np.meshgrid(x, y)
Z = X**2 + Y**2  # f(x,y) 在网格上的值

dZ_dy, dZ_dx = np.gradient(Z, y, x)  # 注意顺序：第一个轴是 y！

print(f"Z.shape: {Z.shape}")
print(f"∂Z/∂x at (25,25) (原点附近): {dZ_dx[25,25]:.3f} (期望 ~0)")
print(f"∂Z/∂y at (0,0) (角落): {dZ_dy[0,0]:.3f} (期望 ~-6)")
```

### 11.3　等高线图——梯度指向"最陡上坡"

📐 **关键直觉**：梯度向量 ∇f = [∂f/∂x, ∂f/∂y] 指向函数值上升最快的方向，且垂直于等高线。

💻 **代码　等高线 + 梯度方向**

```python
import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(-3, 3, 30); y = np.linspace(-3, 3, 30)
X, Y = np.meshgrid(x, y)
Z = X**2 + Y**2
dZ_dy, dZ_dx = np.gradient(Z, y, x)

fig, ax = plt.subplots(figsize=(6, 5))
cs = ax.contour(X, Y, Z, levels=15, cmap='viridis')
ax.clabel(cs, fontsize=7)
# 每 5 个点画一个梯度箭头
ax.quiver(X[::5,::5], Y[::5,::5],
          -dZ_dx[::5,::5], -dZ_dy[::5,::5],
          color='red', alpha=0.6, label='-∇f (最陡下降)')
ax.set_xlabel('x'); ax.set_ylabel('y')
ax.set_title('等高线 + 负梯度方向（指向最小值）')
ax.legend(); ax.set_aspect('equal'); plt.show()
```

负梯度 `-∇f` 指向最陡下降方向——这就是梯度下降的核心几何直觉。
