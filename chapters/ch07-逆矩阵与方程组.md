### 7.1　单位矩阵与逆矩阵——"什么也不做"和"撤销操作"

📐 **定义　单位矩阵（Identity Matrix）**：主对角线全 1、其余全 0 的方阵，记作 **I**。`**A** @ **I** = **I** @ **A** = **A**`。NumPy 中用 `np.eye(n)` 创建。

📐 **定义　逆矩阵（Inverse Matrix）**：方阵 **A** 的逆记作 **A**⁻¹，满足 `**A** @ **A**⁻¹ = **A**⁻¹ @ **A** = **I**`。几何上，**A** 把空间"扭曲"，**A**⁻¹ 把空间"扭曲回来"。

💻 **代码　创建和验证逆矩阵**

```python
import numpy as np

A = np.array([[4, 7],
              [2, 6]])
A_inv = np.linalg.inv(A)

print("A:")
print(A)
print("\nA 的逆:")
print(A_inv)

# 验证：A @ A_inv = I
I_check = A @ A_inv
print("\nA @ A_inv (应为 I):")
print(I_check)
print(f"是否为单位矩阵: {np.allclose(I_check, np.eye(2))}")  # True
```

### 7.2　用逆矩阵解线性方程组

经典问题：2x + 3y = 8, 5x + 4y = 13。写成矩阵形式：`**A** @ **x** = **b**`，其中 **x** = [x, y]ᵀ。等式两边左乘 **A**⁻¹：**x** = **A**⁻¹ @ **b**。

💻 **代码　从方程组到矩阵求解**

```python
import numpy as np

# 方程组: 2x + 3y = 8, 5x + 4y = 13
A = np.array([[2, 3],
              [5, 4]])
b = np.array([8, 13])

# 方法 1: 逆矩阵
x_via_inv = np.linalg.inv(A) @ b
print(f"逆矩阵法: x = {x_via_inv[0]:.2f}, y = {x_via_inv[1]:.2f}")

# 方法 2: np.linalg.solve（更稳定，推荐）
x_via_solve = np.linalg.solve(A, b)
print(f"solve 法:  x = {x_via_solve[0]:.2f}, y = {x_via_solve[1]:.2f}")

# 验证：代回原方程
print(f"2x+3y = {2*x_via_solve[0] + 3*x_via_solve[1]:.1f} (期望 8)")
print(f"5x+4y = {5*x_via_solve[0] + 4*x_via_solve[1]:.1f} (期望 13)")
```

### 7.3　什么时候不可逆？——奇异矩阵

如果一个矩阵"压扁"了空间（降低了维度），它就不可逆。例如，矩阵的两行/列线性相关（一行是另一行的倍数），行列式为 0。

💻 **代码　奇异矩阵与伪逆**

```python
import numpy as np

# 奇异矩阵：两行线性相关（第 2 行 = 第 1 行 × 2）
A_singular = np.array([[1, 2],
                       [2, 4]])  # 行列式 = 1*4 - 2*2 = 0

try:
    np.linalg.inv(A_singular)
except np.linalg.LinAlgError as e:
    print(f"不可逆: {e}")

# 兜底方案: Moore-Penrose 伪逆（最小二乘意义下的最优近似）
A_pinv = np.linalg.pinv(A_singular)
print(f"\n伪逆:\n{A_pinv}")
print(f"A @ pinv @ A ≈ A: {np.allclose(A_singular @ A_pinv @ A_singular, A_singular)}")
```

### 7.4　线性回归闭式解

给定数据 **X** 和标签 **y**，线性回归的目标是找到 **w** 使得 `**X** @ **w** ≈ **y**`。闭式解（正规方程，Normal Equation）：

> **w** = (**X**ᵀ**X**)⁻¹ **X**ᵀ **y**

不需要迭代训练，一行代码解出最优权重。

💻 **代码　手写线性回归闭式解**

```python
import numpy as np

# 生成模拟数据：y = 3*x1 + 2*x2 + 1 + noise
np.random.seed(42)
N = 100
X = np.random.randn(N, 2)           # 两个特征
true_w = np.array([3.0, 2.0])
true_b = 1.0
y = X @ true_w + true_b + np.random.randn(N) * 0.5

# 添加偏置列（截距项）：X → [X | 1]
X_aug = np.column_stack([X, np.ones(N)])  # (100, 3)

# 闭式解: w = (XᵀX)⁻¹ Xᵀ y
XTX = X_aug.T @ X_aug           # (3, 3)
XTX_inv = np.linalg.inv(XTX)    # (3, 3)
w_closed = XTX_inv @ X_aug.T @ y  # (3,)

print(f"闭式解: w1={w_closed[0]:.3f}, w2={w_closed[1]:.3f}, b={w_closed[2]:.3f}")
print(f"真实值: w1={true_w[0]}, w2={true_w[1]}, b={true_b}")
# 应接近 [3.0, 2.0, 1.0]

# 预测
y_pred = X_aug @ w_closed
mse = np.mean((y - y_pred) ** 2)
print(f"MSE: {mse:.4f}")
```

🔗 **AI 连接**：闭式解在特征数少时（< 1000）一步到位，但深度学习百万级参数只能靠梯度下降（第 24 章）迭代逼近。不过，闭式解提供了理解"最优解"的几何直觉，也是第 26 章正规方程推导的前置知识。
