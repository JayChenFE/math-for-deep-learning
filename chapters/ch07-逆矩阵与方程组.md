## 第7章　矩阵的逆与线性方程组 —— 解方程的艺术

> 本章目标：理解逆矩阵的几何含义（"撤销变换"），用 NumPy 求解线性方程组和线性回归闭式解，识别奇异矩阵并学会用伪逆兜底。
> 前置知识：第 6 章（矩阵乘法）

---

### 7.1　单位矩阵与逆矩阵

📐 **定义　单位矩阵（Identity Matrix）**：主对角线全 1、其余全 0 的方阵，记作 **I**。`**A** @ **I** = **I** @ **A** = **A**`。

📐 **定义　逆矩阵（Inverse Matrix）**：`**A** @ **A**⁻¹ = **A**⁻¹ @ **A** = **I**`。几何上，**A**⁻¹ 撤销 **A** 的变换。

💻 **代码　验证逆矩阵**

```python
import numpy as np

A = np.array([[4, 7], [2, 6]])
A_inv = np.linalg.inv(A)
print("A:\n", A)
print("A_inv:\n", A_inv)
print("A @ A_inv:\n", A @ A_inv)
print(f"是否为单位矩阵: {np.allclose(A @ A_inv, np.eye(2))}")  # True
```

---

### 7.2　用逆矩阵解方程组

`2x+3y=8, 5x+4y=13` → `**A** @ **x** = **b**` → `**x** = **A**⁻¹ @ **b**`

💻 **代码　解方程组**

```python
A = np.array([[2, 3], [5, 4]]); b = np.array([8, 13])
x_sol = np.linalg.solve(A, b)  # 比 inv 更稳定
print(f"x={x_sol[0]:.1f}, y={x_sol[1]:.1f}")
print(f"验证: 2x+3y={2*x_sol[0]+3*x_sol[1]:.0f}, 5x+4y={5*x_sol[0]+4*x_sol[1]:.0f}")
```

---

### 7.3　奇异矩阵与伪逆

矩阵两行线性相关（如第 2 行 = 第 1 行 × 2）时行列式为 0，不可逆。

💻 **代码　伪逆兜底**

```python
A_sing = np.array([[1, 2], [2, 4]])  # 行列式 = 0
try: np.linalg.inv(A_sing)
except np.linalg.LinAlgError as e: print(f"不可逆: {e}")

A_pinv = np.linalg.pinv(A_sing)  # Moore-Penrose 伪逆
print(f"伪逆:\n{A_pinv}")
print(f"A @ pinv @ A ≈ A: {np.allclose(A_sing @ A_pinv @ A_sing, A_sing)}")
```

### 7.4　线性回归闭式解

📐 **正规方程**：`**w** = (**X**ᵀ**X**)⁻¹ **X**ᵀ **y**`。无需迭代，一行解出最优权重。

💻 **代码　手写闭式解**

```python
import numpy as np

np.random.seed(42)
N = 100
X = np.random.randn(N, 2)
true_w, true_b = np.array([3.0, 2.0]), 1.0
y = X @ true_w + true_b + np.random.randn(N) * 0.3

X_aug = np.column_stack([X, np.ones(N)])           # (100, 3)
w = np.linalg.inv(X_aug.T @ X_aug) @ X_aug.T @ y   # (3,)
print(f"w1={w[0]:.3f} w2={w[1]:.3f} b={w[2]:.3f}  (真实: 3.0, 2.0, 1.0)")
print(f"MSE: {np.mean((y - X_aug @ w)**2):.4f}")
```
