## 第10章　导数 —— 斜率的"瞬时版本"

> 本章目标：从第 2 章的数值直觉升级到符号公式，掌握常用函数的解析导数（幂函数、指数、sigmoid、ReLU），并用数值微分验证每一个结果。
> 前置知识：第 2 章（数值微分直觉）、第 1 章（Python 函数）

### 10.1　数值微分回顾

📐 **定义　数值微分（Numerical Differentiation）**：`f'(x) ≈ (f(x+h) − f(x)) / h`，h 取 1e-5~1e-8。第 2 章的核心结论：h 太大粗糙，太小舍入误差。

💻 **代码　通用数值微分函数**

```python
import numpy as np

def numerical_derivative(f, x, h=1e-5):
    """对任意函数 f 在点 x 处求数值导数"""
    return (f(x + h) - f(x - h)) / (2 * h)  # 中心差分，比单侧精度高
```

---

### 10.2~10.3　验证幂函数与指数函数

💻 **代码　验证 f'(xⁿ)=n·xⁿ⁻¹ 和 f'(eˣ)=eˣ**

```python
import numpy as np

def numerical_derivative(f, x, h=1e-5):
    return (f(x + h) - f(x - h)) / (2 * h)

# 幂函数 x³ 在 x=2
f_cube = lambda x: x**3
x0 = 2.0
num_d = numerical_derivative(f_cube, x0)
true_d = 3 * x0**2  # 12
print(f"x³ at x=2: 数值={num_d:.6f}  解析={true_d}  误差={abs(num_d-true_d):.2e}")

# 指数函数 eˣ 在 x=1
f_exp = np.exp
num_e = numerical_derivative(f_exp, 1.0)
print(f"eˣ at x=1: 数值={num_e:.6f}  解析={np.e:.6f}  误差={abs(num_e-np.e):.2e}")
```

### 10.4　数值微分的通用性

数值微分不依赖公式——任何可计算函数都能求。这是深度学习梯度检验（gradient check）的基础。

---

### 10.5　二阶导数——变化的加速度

📐 **定义　二阶导数（Second Derivative）**：`f''(x)` = 导数的导数，描述"变化率的变化率"。f''>0 凹向上（凸函数），f''<0 凹向下。

💻 **代码　数值二阶导数**

```python
def numerical_second_derivative(f, x, h=1e-4):
    return (f(x + h) - 2*f(x) + f(x - h)) / (h**2)

f_cube = lambda x: x**3
print(f"x³ at x=2: 数值二阶={numerical_second_derivative(f_cube, 2.0):.3f}  解析={6*2:.1f}")
# f''(x³) = 6x, at x=2 → 12
```

### 10.6　常用函数的解析导数速览

| 函数 | 导数 | 用途 |
|---|---|---|
| `xⁿ` | `n·xⁿ⁻¹` | 多项式 |
| `eˣ` | `eˣ` | 指数 |
| `ln(x)` | `1/x` | 对数 |
| `σ(x)=1/(1+e⁻ˣ)` | `σ(x)(1−σ(x))` | sigmoid 激活 |
| `ReLU(x)=max(0,x)` | `0 (x<0), 1 (x>0)` | ReLU 激活 |

💻 **代码　批量验证**

```python
import numpy as np

def num_deriv(f, x, h=1e-5):
    return (f(x+h) - f(x-h)) / (2*h)

tests = [
    ("x^4",  lambda x: x**4,    2.0, 4*2**3),
    ("ln(x)", lambda x: np.log(x), 2.0, 1/2),
    ("sigmoid", lambda x: 1/(1+np.exp(-x)), 0.0, 0.25),
]
for name, f, x, true_d in tests:
    nd = num_deriv(f, x)
    print(f"{name:10s} at x={x}: 数值={nd:.6f}  解析={true_d:.6f}  ✓")

# ReLU 在 x=0 处导数不存在（左导≠右导）——这是数值微分的盲区
relu = lambda x: max(0, x)
print(f"ReLU at x=0.001:  {num_deriv(relu, 0.001):.1f}")
print(f"ReLU at x=-0.001: {num_deriv(relu, -0.001):.1f}")
print("⚠️ ReLU at x=0: 不可导（深度学习用次梯度 0 代替）")
```

🔗 **AI 连接**：这些解析导数是第 28 章手写反向传播的"本地梯度"来源——ReLU 的梯度是 0 或 1，sigmoid 的梯度是 σ(1−σ)。PyTorch 的 autograd 内部就用这些公式自动计算。
