## 第25章　权重初始化 —— 起点决定终点

> 本章目标：理解全零初始化的对称性问题，掌握 Xavier 和 Kaiming 初始化的原理。
> 前置知识：第 6 章（全连接层）、第 17 章（方差）

---

📐 **全零初始化**：所有神经元更新相同 → 对称性不打破 → 等于只有一个神经元。
📐 **Xavier**：保持前向/反向方差一致，适用于 sigmoid/tanh。
📐 **Kaiming（He）**：针对 ReLU 修正（ReLU 将一半输入清零，需要放大初始方差补偿）。

💻 **代码　不同初始化的输出分布**

```python
import numpy as np

def linear_forward(x, W): return x @ W.T

x = np.random.randn(1000, 256)
for name, std in [("Xavier", np.sqrt(2/(256+256))), ("Kaiming", np.sqrt(2/256)), ("Bad(0.01)", 0.01)]:
    W = np.random.randn(256, 256) * std
    h = linear_forward(x, W)
    print(f"{name:12s}: output mean={h.mean():.4f}, std={h.std():.4f}")
# Xavier: std ≈ 1（保持稳定）
# Kaiming: std ≈ √2（适合 ReLU）
# Bad: std ≪ 1（信号消失）
```

🔗 **AI 连接**：PyTorch `nn.Linear` 默认使用 Kaiming 初始化。`torch.nn.init.kaiming_uniform_` 是训练前的标准操作。
