## 第25章　权重初始化 —— 起点决定终点

> 本章目标：理解全零初始化的对称性陷阱（所有神经元更新完全相同），随机初始化的方差漂移（前向传播中激活值逐层爆炸或消失），以及 Xavier 和 Kaiming 初始化如何精确控制方差保持——让 100 层网络的激活值既不爆炸也不消失。
> 前置知识：第 15 章（正态/均匀分布）、第 17 章（方差）、第 21 章（归一化）

---

### 25.1　全零初始化 —— 对称性灾难

如果把所有权重初始化为 0，每个神经元做完全相同的计算 → 梯度完全相同 → 更新完全相同 → 无论多少层，所有神经元永远一样。网络退化为"只有一个神经元"——对称性扼杀了多样性。

---

### 25.2~25.3　随机初始化 + Xavier —— 方差的保持

随机初始化解决了对称性，但引入了方差漂移：如果初始权重太小，经过 100 层变换后信号衰减到 0（消失）；太大则信号爆炸到 Inf。

**Xavier 初始化**的目标：让每层输出的方差等于输入的方差。对于 `fan_in` 个输入、`fan_out` 个输出的线性层，从 `Uniform[-limit, limit]` 采样，其中 `limit = sqrt(6 / (fan_in + fan_out))`。

---

### 25.4　Kaiming 初始化 —— ReLU 的专属方案

ReLU 把一半的输入（负值）清零了——这相当于"损失了一半的方差"。Kaiming 初始化对此做了补偿：`std = sqrt(2 / fan_in)`——比 Xavier 大 sqrt(2) 倍。PyTorch 的 `nn.Linear` 默认就是 `kaiming_uniform_`。

💻 **代码　三种初始化下 100 层网络的激活值方差变化**

```python
import numpy as np
import matplotlib.pyplot as plt

np.random.seed(42)
n_layers = 100
hidden_dim = 256
batch_size = 512

init_methods = {
    "Xavier": lambda din, dout: np.random.uniform(
        -np.sqrt(6/(din+dout)), np.sqrt(6/(din+dout)), (din, dout)),
    "Kaiming": lambda din, dout: np.random.randn(din, dout) * np.sqrt(2/din),
    "Small Gauss": lambda din, dout: np.random.randn(din, dout) * 0.01,
    "Large Gauss": lambda din, dout: np.random.randn(din, dout) * 1.0,
}

fig, axes = plt.subplots(1, 4, figsize=(16, 3.5))

for ax, (name, init_fn) in zip(axes, init_methods.items()):
    x = np.random.randn(batch_size, hidden_dim)
    stds = [x.std()]
    for _ in range(n_layers):
        W = init_fn(hidden_dim, hidden_dim)
        x = np.maximum(0, x @ W)  # ReLU
        stds.append(x.std())
    ax.plot(range(n_layers + 1), stds, linewidth=1.5)
    ax.axhline(y=1.0, color='red', linestyle='--', alpha=0.5)
    ax.set_title(f'{name}\n(final std={stds[-1]:.4f})')
    ax.set_xlabel('层数'); ax.set_ylabel('激活值标准差')

plt.suptitle('100层网络激活值标准差变化', fontsize=13)
plt.tight_layout(); plt.show()
print("Kaiming: 标准差始终保持 ~1 — 100层后信号不衰减")
print("Xavier: ReLU后逐渐衰减 (未补偿ReLU的'减半'效应)")
print("Small/Large: 迅速归零或爆炸")
```

---

### 25.5　总结

| 初始化 | 适用激活函数 | 核心公式 |
|--------|------------|---------|
| Xavier | tanh/sigmoid | `limit = sqrt(6/(fan_in+fan_out))` |
| Kaiming | ReLU/LeakyReLU | `std = sqrt(2/fan_in)` |
