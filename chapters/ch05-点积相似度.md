> 本章目标：掌握点积的代数计算和几何含义，理解余弦相似度及其在推荐系统和 Transformer 注意力机制中的应用。核心公式：`Q @ Kᵀ` 就是批量点积。

### 5.1　代数定义——对应相乘再求和

📐 **定义　点积（Dot Product）**：`**a**·**b** = Σ aᵢ·bᵢ`。结果是一个**标量**。NumPy 中用 `np.dot(a, b)` 或 `a @ b`。

💻 **代码　点积的手工实现与验证**

```python
import numpy as np

a = np.array([2, 3, 1])
b = np.array([4, 0, 5])

dot_manual = sum(a[i] * b[i] for i in range(len(a)))
print(f"手工: {dot_manual}")                    # 13
print(f"np.dot: {np.dot(a, b)}")                # 13
print(f"a @ b: {a @ b}")                        # 13
print(f"(a * b).sum(): {(a * b).sum()}")        # 13
```

---

### 5.2　几何本质——方向匹配度的标尺

📐 **几何公式**：`**a**·**b** = ‖**a**‖·‖**b**‖·cos(θ)`。

- `**a**·**b** > 0` → 夹角 < 90°（方向大致相同）
- `**a**·**b** = 0` → 夹角 = 90°（**正交**——"不相关"）
- `**a**·**b** < 0` → 夹角 > 90°（方向大致相反）

📐 **定义　余弦相似度（Cosine Similarity）**：`cos_sim = (**a**·**b**) / (‖**a**‖·‖**b**‖)`。值域 [−1, 1]，只衡量方向，不关心长度。

💻 **代码　验证代数 = 几何**

```python
import numpy as np

a = np.array([3., 0.]); b = np.array([1.5, 1.5])
dot_alg = np.dot(a, b)
dot_geo = np.linalg.norm(a) * np.linalg.norm(b) * np.cos(np.deg2rad(45))
print(f"代数: {dot_alg:.4f}  几何: {dot_geo:.4f}  相等✅")

# 三种典型情况
for v1, v2, desc in [([1,0],[1,0],"同向cos=1"), ([1,0],[0,1],"正交cos=0"), ([1,0],[-1,0],"反向cos=-1")]:
    v1, v2 = np.array(v1,float), np.array(v2,float)
    cs = np.dot(v1,v2)/(np.linalg.norm(v1)*np.linalg.norm(v2))
    print(f"{desc}: cos={cs:.0f}")
```

### 5.3　推荐系统——点积找最匹配

💻 **代码　简易推荐系统**

```python
import numpy as np

user = np.array([0.2, 0.9, 0.1, 0.7])  # 用户偏好
items = {"键盘":[0,1,0,0.8], "运动鞋":[0.5,0,0.9,0.3], "耳机":[0.1,0.9,0.2,0.6],
         "帆布袋":[0.9,0,0.6,0.1], "显示器":[0,1,0,0.9]}

def cosine_sim(a, b):
    return np.dot(a,b)/(np.linalg.norm(a)*np.linalg.norm(b)+1e-8)

scores = {n: cosine_sim(user, np.array(v)) for n, v in items.items()}
for i, (name, s) in enumerate(sorted(scores.items(), key=lambda x: -x[1]), 1):
    print(f"  {i}. {name}: {s:.4f} {'█'*int(s*30)}")
```

### 5.4　批量点积——Q@Kᵀ 的原型

💻 **代码　一次性计算所有 (查询, 键) 对**

```python
import numpy as np

Q = np.random.randn(4, 3)   # 4个查询
K = np.random.randn(6, 3)   # 6个键
scores = Q @ K.T             # (4, 6) —— 所有对的点积
print(f"scores.shape: {scores.shape}")
print(f"用户0对6个物品的打分: {scores[0]}")

top2 = np.argsort(scores, axis=1)[:, -2:]
print(f"每个用户top-2物品:\n{top2}")
```

🔗 **AI 连接**：这就是 `Q @ K.transpose(-2, -1)`。第 5 章你已经掌握了第 29 章注意力机制的核心运算。
