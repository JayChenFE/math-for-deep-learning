## 第24章　优化算法：从 SGD 到 AdamW

> 本章目标：理解 SGD+Momentum、Adam、AdamW 的核心原理，掌握学习率 warmup+cosine decay 调度。
> 前置知识：第 12 章（梯度下降）

---

### SGD + Momentum

📐 SGD: `θ -= lr * ∇L`。Momentum: `v = β*v + ∇L; θ -= lr * v`——给梯度加惯性，小球滚下山。

### Adam

📐 Adam = Momentum + 自适应学习率。稀疏参数大步走，活跃参数小步走。是训练 Transformer 的事实标准。

### AdamW

📐 把 Weight Decay 从 L2 正则化中解耦：`θ *= (1 - lr * λ)`，然后做 Adam 更新。HuggingFace Trainer 默认用 AdamW。

### 学习率调度

📐 **Warmup**：前 N 步 lr 从 0 线性增长 → 防止训练初期不稳定。
📐 **Cosine Decay**：从目标 lr 余弦衰减到 0 → 平滑收束。

💻 **代码　优化器对比**

```python
import numpy as np

def f(x): return x[0]**2 + 10*x[1]**2  # 狭长峡谷

def sgd_step(x, lr=0.03):
    g = np.array([2*x[0], 20*x[1]]); return x - lr*g

def momentum_step(x, v, lr=0.03, beta=0.9):
    g = np.array([2*x[0], 20*x[1]]); v = beta*v + g; return x - lr*v, v

x_sgd = np.array([3.0, 2.0]); x_mom = x_sgd.copy(); v = np.zeros(2)
for t in range(100):
    x_sgd = sgd_step(x_sgd)
    x_mom, v = momentum_step(x_mom, v)
print(f"SGD final: f={f(x_sgd):.4f}  Momentum final: f={f(x_mom):.4f}")
```
