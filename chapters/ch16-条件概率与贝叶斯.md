## 第16章　条件概率与贝叶斯定理 —— 已知信息改变判断

> 本章目标：理解条件概率 P(A|B) 和贝叶斯公式，用代码模拟"检测阳性后真实患病率"的经典反直觉案例。
> 前置知识：第 15 章（概率分布）

---

### 16.1　条件概率

📐 **定义　条件概率（Conditional Probability）**：P(A|B) = P(A∩B) / P(B)。已知 B 发生，A 发生的概率。

### 16.2　贝叶斯公式

📐 **贝叶斯定理**：P(A|B) = P(B|A)·P(A) / P(B)。将"先验信念" P(A) 用新证据 B 更新为"后验" P(A|B)。

经典案例：检测准确率 99%，发病率 1%，阳性后真实患病率仅约 50%——因为基数大，假阳性人数远超真阳性。

💻 **代码　贝叶斯更新模拟**

```python
# 参数
p_disease = 0.01       # 先验：发病率 1%
p_pos_given_disease = 0.99   # 灵敏度
p_pos_given_healthy = 0.01   # 假阳性率

# 贝叶斯计算
p_pos = p_pos_given_disease * p_disease + p_pos_given_healthy * (1 - p_disease)
p_disease_given_pos = p_pos_given_disease * p_disease / p_pos
print(f"P(患病|阳性) = {p_disease_given_pos:.1%}")
# ~50% —— 远低于直观猜测的 99%
```

### 16.3　朴素贝叶斯分类器

📐 **朴素贝叶斯**：假设所有特征独立（"朴素"），用贝叶斯公式计算每个类别的后验概率，选最大者。垃圾邮件分类的经典基线模型。

💻 **代码　简易概率分类**

```python
import numpy as np

# 模拟：两类别（0/1），前一特征出现频率
# 类别0: 特征A出现率0.3, 特征B出现率0.2, 先验0.5
# 类别1: 特征A出现率0.8, 特征B出现率0.7, 先验0.5
def naive_bayes(features):
    p0 = 0.5 * (0.3 if features[0] else 0.7) * (0.2 if features[1] else 0.8)
    p1 = 0.5 * (0.8 if features[0] else 0.2) * (0.7 if features[1] else 0.3)
    return 0 if p0 > p1 else 1

print(f"[1,1] → class {naive_bayes([1, 1])}")  # 预期 class 1
print(f"[0,0] → class {naive_bayes([0, 0])}")  # 预期 class 0
```
