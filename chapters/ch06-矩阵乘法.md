### 6.1　矩阵与向量相乘——线性层核心

📐 **定义　矩阵-向量乘法**：`**y** = **W** @ **x**`。`(m,n) @ (n,) → (m,)`。y 的每个分量是 W 对应行与 x 的点积。这就是 `nn.Linear(d_in, d_out)` 的数学本质。

💻 **代码　手写矩阵-向量乘法**

```python
import numpy as np

W = np.array([[1, 2, 0, -1], [0, 1, 3, 2], [2, 0, 1, 0]])  # (3,4)
x = np.array([2, 1, 3, 4])                                    # (4,)
y = W @ x  # (3,)
print(f"W @ x = {y}")

for i in range(3):
    assert abs(np.dot(W[i], x) - y[i]) < 1e-10
print("矩阵-向量乘法 = 逐行点积 ✅")
```

### 6.2　矩阵与矩阵乘法——批量变换

📐 **定义　矩阵-矩阵乘法**：`**C** = **A** @ **B**`。`(m,k) @ (k,n) → (m,n)`。C[i,j] = A 第 i 行 · B 第 j 列。

💻 **代码　三重循环 vs NumPy**

```python
import numpy as np, time

A = np.random.randn(200, 300); B = np.random.randn(300, 400)

def slow_matmul(A, B):
    m, k = A.shape; _, n = B.shape
    C = np.zeros((m, n))
    for i in range(m):
        for j in range(n):
            for t in range(k): C[i,j] += A[i,t] * B[t,j]
    return C

t0 = time.perf_counter(); C_slow = slow_matmul(A, B); t1 = time.perf_counter()
C_fast = A @ B; t2 = time.perf_counter()
print(f"三重循环: {t1-t0:.2f}s  NumPy @: {t2-t1:.4f}s  加速: {(t1-t0)/(t2-t1):.0f}x")
assert np.allclose(C_slow, C_fast)
```

### 6.3　几何视角——矩阵扭曲空间

💻 **代码　网格变形可视化**

```python
import numpy as np
import matplotlib.pyplot as plt

def plot_grid_transform(W, ax, title=""):
    pts = np.stack(np.meshgrid(np.linspace(-2,2,10), np.linspace(-2,2,10)), axis=-1)
    orig = pts.reshape(-1, 2)            # (100, 2)
    trans = orig @ W.T                   # (100, 2)
    ax.scatter(orig[:,0], orig[:,1], c='blue', alpha=0.3, s=10, label='original')
    ax.scatter(trans[:,0], trans[:,1], c='red', alpha=0.6, s=10, label='transformed')
    ax.axhline(0,color='gray',lw=0.5); ax.axvline(0,color='gray',lw=0.5)
    ax.set_xlim(-5,5); ax.set_ylim(-5,5); ax.set_aspect('equal'); ax.legend(fontsize=8)
    ax.set_title(title)

fig, axes = plt.subplots(1, 3, figsize=(15, 4))
plot_grid_transform(np.eye(2), axes[0], "Identity")
plot_grid_transform(np.array([[2,0],[0,1]]), axes[1], "Stretch x")
plot_grid_transform(np.array([[1,0.5],[0,1]]), axes[2], "Shear")
plt.tight_layout(); plt.show()
```

🔗 **AI 连接**：神经网络的每一层就是一次空间扭曲。多层叠加把原始数据映射到线性可分的空间（第 28 章可视化）。

### 6.4　全连接层手撕

💻 **代码　手写 nn.Linear**

```python
import numpy as np

def linear_layer(X, W, b):
    """X:(batch,d_in) W:(d_out,d_in) b:(d_out,) → (batch,d_out)"""
    return X @ W.T + b  # b 自动广播到每行

batch, d_in, d_out = 32, 784, 256
X = np.random.randn(batch, d_in)
W = np.random.randn(d_out, d_in) * 0.01
b = np.zeros(d_out)
out = linear_layer(X, W, b)
print(f"输入 {X.shape} → 输出 {out.shape}")
assert out.shape == (batch, d_out)
assert np.allclose(out[0], X[0] @ W.T + b)
print("全连接层 ✅")
```

### 6.5　广播机制精讲

📐 **定义　广播（Broadcasting）**：不同形状数组运算时，从最后一维往前对齐，维度相等或一方为 1 即可广播。`(32,784) @ (784,256) + (256,)` 中 `(256,)` 自动扩为 `(32,256)`。

💻 **代码　广播规则**

```python
import numpy as np

A = np.ones((3, 4)); v = np.array([10, 20, 30, 40])
print(f"(3,4)+(4,) = {(A+v).shape}")  # (3,4)

col = np.array([[1],[2],[3]]); row = np.array([10,20,30,40])
print(f"(3,1)+(4,) = {(col+row).shape}")  # (3,4)
print(col + row)

try: A + np.array([1,2,3])
except ValueError as e: print(f"广播失败: {e}")
```

🔗 **AI 连接**：Transformer 中 `(batch,heads,seq,d_k)` 的 attention 与 `(seq,seq)` 的 mask 相加——靠的就是广播。
