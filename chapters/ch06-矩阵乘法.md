### 6.1　矩阵与向量相乘——线性层的核心

📐 **定义　矩阵-向量乘法**：`**y** = **W** @ **x**`。一个 `(m, n)` 的矩阵乘以一个 `(n,)` 的向量，得到一个 `(m,)` 的向量。**y** 的每个分量是 **W** 的一行与 **x** 的点积。

这正是 `nn.Linear(d_in, d_out)` 前向传播的数学本质。

💻 **代码　手写矩阵-向量乘法**

```python
import numpy as np

# W: (3, 4) — 3 个输出神经元，4 个输入特征
W = np.array([[1, 2, 0, -1],
              [0, 1, 3,  2],
              [2, 0, 1,  0]])
x = np.array([2, 1, 3, 4])  # (4,) — 输入向量

# NumPy 一行搞定
y = W @ x  # shape: (3,)
print(f"W @ x = {y}")

# 手写验证：y[i] = W 的第 i 行与 x 的点积
for i in range(W.shape[0]):
    manual = np.dot(W[i], x)
    print(f"  第 {i} 行点积: {manual}")
    assert abs(manual - y[i]) < 1e-10
print("矩阵-向量乘法 = 逐行点积 ✅")
```

### 6.2　矩阵与矩阵相乘——批量变换

📐 **定义　矩阵-矩阵乘法**：`**C** = **A** @ **B**`。若 `A.shape = (m, k)`，`B.shape = (k, n)`，则 `C.shape = (m, n)`。`C[i, j] = A` 的第 i 行与 B 的第 j 列的点积。

这是深度学习中运算量最大的操作——全连接层、注意力计算、卷积展开后全是矩阵乘法。

💻 **代码　三重循环 vs NumPy 向量化**

```python
import numpy as np
import time

A = np.random.randn(200, 300)  # (200, 300)
B = np.random.randn(300, 400)  # (300, 400)

# 三重循环（纯 Python——千万别在生产代码中用）
def slow_matmul(A, B):
    m, k = A.shape
    _, n = B.shape
    C = np.zeros((m, n))
    for i in range(m):
        for j in range(n):
            s = 0
            for t in range(k):
                s += A[i, t] * B[t, j]
            C[i, j] = s
    return C

t0 = time.perf_counter()
C_slow = slow_matmul(A, B)
t1 = time.perf_counter()

C_fast = A @ B
t2 = time.perf_counter()

print(f"三重循环: {t1 - t0:.3f} 秒")
print(f"NumPy @:  {t2 - t1:.4f} 秒")
print(f"加速比: {(t1 - t0) / (t2 - t1):.0f}x")
assert np.allclose(C_slow, C_fast)
# NumPy 底层调用 BLAS，利用 CPU SIMD + 多级缓存 + 多线程
```

### 6.3　几何视角——矩阵将空间"扭曲"

一个 `(2, 2)` 矩阵乘以一个 2D 向量，相当于把这个向量从一个位置"映射"到另一个位置。把平面上所有网格点都做这个变换，就能直观看到矩阵如何扭曲空间。

💻 **代码　网格变形——可视化线性变换**

```python
import numpy as np
import matplotlib.pyplot as plt

def plot_grid_transform(W, ax, title=""):
    """画出矩阵 W 对 2D 空间网格的变换效果"""
    # 原始网格
    points = np.stack(np.meshgrid(np.linspace(-2, 2, 10),
                                   np.linspace(-2, 2, 10)), axis=-1)
    original = points.reshape(-1, 2)        # (100, 2)
    transformed = original @ W.T            # (100, 2)

    # 画变换前后的网格点
    ax.scatter(original[:, 0], original[:, 1], c='blue', alpha=0.3, s=10, label='原始')
    ax.scatter(transformed[:, 0], transformed[:, 1], c='red', alpha=0.6, s=10, label='变换后')
    ax.axhline(0, color='gray', linewidth=0.5)
    ax.axvline(0, color='gray', linewidth=0.5)
    ax.set_xlim(-5, 5); ax.set_ylim(-5, 5)
    ax.set_aspect('equal'); ax.legend(fontsize=8)
    ax.set_title(title)

# 三种典型变换
fig, axes = plt.subplots(1, 3, figsize=(15, 4))
plot_grid_transform(np.array([[1, 0], [0, 1]]),  axes[0], "单位矩阵（不变）")
plot_grid_transform(np.array([[2, 0], [0, 1]]),  axes[1], "x 方向拉伸 2×")
plot_grid_transform(np.array([[1, 0.5], [0, 1]]), axes[2], "剪切变换")
plt.tight_layout(); plt.show()
```

🔗 **AI 连接**：神经网络的每一层就是一次空间扭曲。多层叠加 = 多次连续扭曲，最终把原始数据（不可线性分开）映射到一个可以被简单分类器切分的空间（第 28 章将可视化这个过程）。

### 6.4　全连接层手撕

把前六章的知识组装起来：`output = input @ weight.T + bias` 就是一个完整的全连接层。

💻 **代码　手写 nn.Linear 前向传播**

```python
import numpy as np

def linear_layer(X, W, b):
    """
    X: (batch, d_in)    — 批量输入
    W: (d_out, d_in)    — 权重矩阵
    b: (d_out,)         — 偏置向量
    返回: (batch, d_out)
    """
    return X @ W.T + b  # 广播：b 自动扩展到每一行

# 模拟数据
batch_size, d_in, d_out = 32, 784, 256
X = np.random.randn(batch_size, d_in)
W = np.random.randn(d_out, d_in) * 0.01  # 小随机初始化
b = np.zeros(d_out)

output = linear_layer(X, W, b)
print(f"输入:  {X.shape}")      # (32, 784)
print(f"权重:  {W.shape}")      # (256, 784)
print(f"输出:  {output.shape}") # (32, 256)

# 验证第 0 个样本的输出 = X[0] @ W.T + b
sample_out = X[0] @ W.T + b
assert np.allclose(output[0], sample_out)
print("全连接层验证通过 ✅")
```

### 6.5　广播机制精讲

📐 **定义　广播（Broadcasting）**：当两个形状不同的数组运算时，NumPy/PyTorch 自动将较小数组沿缺失维度"复制"到匹配形状。规则：从最后一维往前对齐，维度相等或有一方为 1 即可广播。

`(32, 784) @ (784, 256) + (256,)` 中，`(256,)` 被广播为 `(1, 256)` 再复制到 `(32, 256)`。这是读懂 PyTorch 代码的基本功。

💻 **代码　广播规则演示**

```python
import numpy as np

# 案例 1: (3, 4) + (4,) → (3, 4)
A = np.ones((3, 4))
v = np.array([10, 20, 30, 40])
print(f"(3,4) + (4,) = {(A + v).shape}")  # (3, 4)

# 案例 2: (3, 1) + (4,) → (3, 4)
col = np.array([[1], [2], [3]])  # (3, 1)
row = np.array([10, 20, 30, 40]) # (4,)
result = col + row
print(f"(3,1) + (4,) = {result.shape}")  # (3, 4)
print(result)

# 案例 3: 无法广播的情况
# (3, 4) + (3,) → 最后一维 4 vs 3，无法兼容 → 报错
try:
    A + np.array([1, 2, 3])
except ValueError as e:
    print(f"广播失败：{e}")
```

🔗 **AI 连接**：Transformer 中 `(batch, heads, seq, d_k)` 的 attention 与 `(seq, seq)` 的 mask 相加——靠的就是广播。不理解广播就理解不了 Transformer 的实现代码。
