### 20.1~20.2　二进制表示与精度

📐 float32（单精度）：1 位符号 + 8 位指数 + 23 位尾数，约 7 位有效数字。
📐 float64（双精度）：1+11+52，约 16 位有效数字。

💻 **代码　精度差异演示**

```python
import numpy as np
# float32: 1 + 1e-8 → 1.0（精度不够）
print(f"float32: {(np.float32(1.0) + np.float32(1e-8)).item():.10f}")
print(f"float64: {(1.0 + 1e-8):.10f}")

# 小梯度时参数不更新
w = np.float32(1.0); grad = np.float32(1e-8); lr = np.float32(0.01)
w_new = w - lr * grad
print(f"w={w}, update={lr*grad}, w_new={w_new}, changed={w != w_new}")
```

### 20.3　float16 vs bfloat16

| 类型 | 范围 | 精度 | 用途 |
|---|---|---|---|
| fp32 | ±3.4e38 | 7 位 | 训练主精度 |
| fp16 | ±65504 | 3 位 | 推理加速 |
| bf16 | ±3.4e38 | 2 位 | 训练加速（安全） |

bf16 和 fp32 范围一样大——不会溢出，是训练的"安全加速器"。

### 20.4　混合精度训练

前向/反向用 fp16 加速，权重副本用 fp32 保证精度。PyTorch 中一行 `torch.cuda.amp.autocast()` 即可启用。
