### 12.1　打包偏导数——梯度向量

📐 **定义　梯度（Gradient）**：∇f = [∂f/∂x₁, ∂f/∂x₂, ..., ∂f/∂xₙ]ᵀ。梯度是一个**向量**，每个分量是函数对该变量的偏导数。

---

### 12.2　梯度方向 = 最陡上升方向

梯度最重要的几何性质：∇f 指向 f 增长最快的方向；−∇f 指向 f 下降最快的方向。

💻 **代码　沿负梯度走一步验证函数值下降**

```python
import numpy as np

def f(x):
    return x[0]**2 + 10*x[1]**2  # 狭长峡谷——SGD 经典测试函数

def gradient(f, x, h=1e-5):
    """对任意 f: R^n→R 求数值梯度"""
    grad = np.zeros_like(x)
    for i in range(len(x)):
        x_plus = x.copy(); x_plus[i] += h
        x_minus = x.copy(); x_minus[i] -= h
        grad[i] = (f(x_plus) - f(x_minus)) / (2*h)
    return grad

x = np.array([3.0, 2.0])
grad_f = gradient(f, x)
print(f"在 x={x} 处: f={f(x):.2f}")
print(f"梯度 ∇f = {grad_f}")

lr = 0.05
x_new = x - lr * grad_f
print(f"沿负梯度走一步: x_new={x_new}, f_new={f(x_new):.2f}")
print(f"f 下降: {f(x) > f(x_new)} ✓")
```

### 12.3　梯度垂直于等高线

💻 **代码　quiver + contour 可视化**

```python
import numpy as np
import matplotlib.pyplot as plt

X, Y = np.meshgrid(np.linspace(-4, 4, 40), np.linspace(-3, 3, 30))
Z = X**2 + 10*Y**2
dZ_dy, dZ_dx = np.gradient(Z, Y[:,0], X[0,:])

fig, ax = plt.subplots(figsize=(7, 5))
ax.contour(X, Y, Z, levels=20, cmap='viridis', alpha=0.5)
skip = 4
ax.quiver(X[::skip,::skip], Y[::skip,::skip],
          -dZ_dx[::skip,::skip], -dZ_dy[::skip,::skip],
          color='red', scale=200, width=0.004, label='-∇f')
ax.set_xlabel('x'); ax.set_ylabel('y')
ax.set_title('负梯度方向总是垂直于等高线，指向谷底')
ax.legend(); ax.set_aspect('equal'); plt.show()
```

### 12.4　AI 核心：损失函数的梯度

📐 **核心公式**：`θ_new = θ_old − lr · ∇L(θ)`。整个深度学习训练就是这个公式的循环——计算损失对每个参数的梯度，沿负梯度方向走一小步。

💻 **代码　一元损失函数的梯度下降**

```python
import numpy as np

L = lambda w: (w - 3)**2  # 最优解 w=3
dL = lambda w: 2*(w - 3)  # 解析梯度

w = 0.0; lr = 0.1
for t in range(30):
    w = w - lr * dL(w)
    if t % 10 == 0:
        print(f"step {t:3d}: w={w:.4f}, L={L(w):.6f}")
print(f"最终 w={w:.4f} (真实最优=3.0)  ✓")
```
