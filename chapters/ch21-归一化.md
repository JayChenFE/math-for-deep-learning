## 第21章　归一化：把数值拉回安全区间

> 本章目标：理解 BatchNorm/LayerNorm/RMSNorm 的原理和差异，手写实现 LayerNorm 和 RMSNorm。
> 前置知识：第 6 章（矩阵）、第 17 章（方差）

---

### LayerNorm —— Transformer 标配

📐 **LayerNorm**：在最后一维内归一化。`y = (x−μ)/σ * γ + β`。γ 和 β 是可学习参数。Transformer 用 LayerNorm 而不是 BatchNorm 因为序列维度可变 + 推理一致性。

### RMSNorm —— LLaMA 的选择

📐 **RMSNorm**：只除 RMS 不减去均值。`y = x / RMS(x) * γ`。计算快约 15%，LLaMA/Mistral 都用它。

💻 **代码　手写 LayerNorm 和 RMSNorm**

```python
import numpy as np

def layer_norm(x, eps=1e-5):
    mean = x.mean(axis=-1, keepdims=True)
    var = x.var(axis=-1, keepdims=True)
    return (x - mean) / np.sqrt(var + eps)

def rms_norm(x, eps=1e-5):
    rms = np.sqrt((x ** 2).mean(axis=-1, keepdims=True))
    return x / (rms + eps)

x = np.random.randn(4, 8) * 5 + 10  # 模拟极端值输入
ln = layer_norm(x); rn = rms_norm(x)
print(f"LayerNorm: mean={ln.mean(axis=-1)[:2].round(3)} (≈0), std={ln.std(axis=-1)[:2].round(3)} (≈1)")
print(f"RMSNorm:   rms={np.sqrt((rn**2).mean(axis=-1))[:2].round(3)} (≈1)")
```
