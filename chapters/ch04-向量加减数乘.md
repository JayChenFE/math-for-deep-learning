### 4.1　向量加法——平移

📐 **定义　向量加法（Vector Addition）**：两个同维度向量对应位置相加。`**a** + **b** = [a₁+b₁, ..., aₙ+bₙ]`。几何：平行四边形法则——以 a、b 为邻边作平行四边形，对角线就是 a+b。

💻 **代码　向量加法的代数与几何**

```python
import numpy as np
import matplotlib.pyplot as plt

a = np.array([2, 1])
b = np.array([1, 3])
c = a + b
print(f"a + b = {c}")  # [3, 4]

# 平行四边形法则可视化
fig, ax = plt.subplots(figsize=(5, 5))
ax.set_xlim(0, 5); ax.set_ylim(0, 5)
ax.axhline(0, color='gray', linewidth=0.5)
ax.axvline(0, color='gray', linewidth=0.5)
ax.quiver(0, 0, a[0], a[1], angles='xy', scale_units='xy', scale=1, color='b', width=0.015, label='a')
ax.quiver(0, 0, b[0], b[1], angles='xy', scale_units='xy', scale=1, color='r', width=0.015, label='b')
ax.quiver(0, 0, c[0], c[1], angles='xy', scale_units='xy', scale=1, color='purple', width=0.015, label='a+b')
ax.plot([a[0], c[0]], [a[1], c[1]], 'r--', alpha=0.5)
ax.plot([b[0], c[0]], [b[1], c[1]], 'b--', alpha=0.5)
ax.legend(); ax.set_aspect('equal')
ax.set_title('a + b = 平行四边形对角线')
plt.show()
```

### 4.2　向量数乘——缩放

📐 **定义　向量数乘（Scalar Multiplication）**：`k·**v** = [k·v₁, ..., k·vₙ]`。k>0 方向不变长度缩放 |k| 倍，k<0 方向反转。深度学习中 `lr * grad` 就是对梯度向量的数乘。

💻 **代码　向量缩放**

```python
import numpy as np
import matplotlib.pyplot as plt

v = np.array([2, 1])
fig, ax = plt.subplots(figsize=(5, 5))
ax.set_xlim(-3, 3); ax.set_ylim(-3, 3)
ax.axhline(0, color='gray', linewidth=0.5)
ax.axvline(0, color='gray', linewidth=0.5)

for k, color, lbl in [(2, 'r', '2v'), (1, 'b', 'v'), (-0.5, 'orange', '-0.5v')]:
    ax.quiver(0, 0, (k*v)[0], (k*v)[1], angles='xy', scale_units='xy', scale=1,
              color=color, width=0.02, label=lbl)
ax.legend(); ax.set_aspect('equal')
ax.set_title('数乘：方向不变，长度缩放 |k| 倍')
plt.show()

print(f"‖v‖={np.linalg.norm(v):.3f}  ‖2v‖={np.linalg.norm(2*v):.3f}  ‖-0.5v‖={np.linalg.norm(-0.5*v):.3f}")
```

### 4.3　向量减法——指向"对面"

📐 **定义　向量减法（Vector Subtraction）**：`**a** − **b** = **a** + (−**b**)`。几何：从 b 终点指向 a 终点的向量。

💻 **代码　减法几何**

```python
import numpy as np
import matplotlib.pyplot as plt

a = np.array([3, 2]); b = np.array([1, 1])
diff = a - b  # [2, 1]——从 b 终点指向 a 终点

fig, ax = plt.subplots(figsize=(5, 5))
ax.set_xlim(0, 5); ax.set_ylim(0, 4)
ax.quiver(0, 0, a[0], a[1], angles='xy', scale_units='xy', scale=1, color='b', width=0.02, label='a')
ax.quiver(0, 0, b[0], b[1], angles='xy', scale_units='xy', scale=1, color='r', width=0.02, label='b')
ax.quiver(b[0], b[1], diff[0], diff[1], angles='xy', scale_units='xy', scale=1,
          color='green', width=0.02, label='a-b')
ax.legend(); ax.set_aspect('equal')
ax.set_title('a - b：从 b 终点指向 a 终点')
plt.show()
```

### 4.4　AI 应用：词嵌入语义运算

> **国王** − **男人** + **女人** ≈ **女王**

本质上就是向量加减法编码了语义变换。

💻 **代码　模拟词嵌入语义运算**

```python
import numpy as np

np.random.seed(42)
king  = np.array([0.5, 0.7, 0.3, 0.2])
man   = np.array([0.3, 0.4, 0.1, 0.1])
woman = np.array([0.2, 0.5, 0.2, 0.2])
queen_true = np.array([0.4, 0.8, 0.4, 0.3])

queen_pred = king - man + woman
cos_sim = np.dot(queen_pred, queen_true) / (np.linalg.norm(queen_pred) * np.linalg.norm(queen_true))
print(f"king - man + woman = {queen_pred}")
print(f"与真实 queen 的余弦相似度: {cos_sim:.4f}")
```

🔗 **AI 连接**：Transformer 的 Self-Attention 本质上是让每个词对所有词做"加权组合"——只是权重不是 ±1，而是动态计算的注意力分数（第 29 章）。
