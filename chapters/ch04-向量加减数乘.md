## 第4章　向量的加减与数乘 —— 平移与缩放

> 本章目标：掌握向量的加法、减法和数乘三种基本运算，并用 `matplotlib.quiver` 将运算结果可视化。理解为什么"国王 − 男人 + 女人 ≈ 女王"这个著名的词嵌入等式本质上是向量的加减法。
> 前置知识：第 1 章（NumPy 数组、matplotlib）、第 3 章（向量概念、shape）

---

### 4.1　向量加法——平移

📐 **定义　向量加法（Vector Addition）**：两个同维度向量对应位置相加。`**a** + **b** = [a₁+b₁, a₂+b₂, ..., aₙ+bₙ]`。几何上，向量加法相当于**平移**：把 **b** 的起点放在 **a** 的终点上，从 **a** 起点指向 **b** 终点的箭头就是 **a**+**b**（平行四边形法则）。

💻 **代码　向量加法的代数与几何**

```python
import numpy as np
import matplotlib.pyplot as plt

a = np.array([2, 1])
b = np.array([1, 3])
c = a + b

print(f"a = {a}")
print(f"b = {b}")
print(f"a + b = {c}")  # [3, 4]

# 几何可视化：平行四边形法则
fig, ax = plt.subplots(figsize=(5, 5))
ax.set_xlim(0, 5); ax.set_ylim(0, 5)
ax.axhline(0, color='gray', linewidth=0.5)
ax.axvline(0, color='gray', linewidth=0.5)

# quiver: 起点(x,y), 方向(dx,dy), angles='xy', scale_units='xy', scale=1
ax.quiver(0, 0, a[0], a[1], angles='xy', scale_units='xy', scale=1,
          color='b', width=0.01, label='a')
ax.quiver(0, 0, b[0], b[1], angles='xy', scale_units='xy', scale=1,
          color='r', width=0.01, label='b')
ax.quiver(0, 0, c[0], c[1], angles='xy', scale_units='xy', scale=1,
          color='purple', width=0.01, label='a + b')
# 画平行四边形
ax.plot([a[0], c[0]], [a[1], c[1]], 'r--', alpha=0.5)
ax.plot([b[0], c[0]], [b[1], c[1]], 'b--', alpha=0.5)

ax.legend(); ax.set_aspect('equal')
ax.set_title('a + b = 平行四边形对角线')
plt.show()
```

### 4.2　向量数乘——缩放

📐 **定义　向量数乘（Scalar Multiplication）**：用一个标量 `k` 乘以向量每个分量。`k·**v** = [k·v₁, k·v₂, ..., k·vₙ]`。效果：**v** 的方向不变（k>0 时），长度缩放 |k| 倍。k<0 时方向反转。

数乘是深度学习中缩放操作的原型：学习率 `lr * grad` 就是对梯度向量的数乘。

💻 **代码　向量缩放**

```python
import numpy as np
import matplotlib.pyplot as plt

v = np.array([2, 1])

fig, ax = plt.subplots(figsize=(5, 5))
ax.set_xlim(-3, 3); ax.set_ylim(-3, 3)
ax.axhline(0, color='gray', linewidth=0.5)
ax.axvline(0, color='gray', linewidth=0.5)

for k, color, label in [(2, 'r', '2v'), (1, 'b', 'v'),
                          (-0.5, 'orange', '-0.5v')]:
    kv = k * v
    ax.quiver(0, 0, kv[0], kv[1], angles='xy', scale_units='xy', scale=1,
              color=color, width=0.02, label=label)

ax.legend(); ax.set_aspect('equal')
ax.set_title('数乘：方向(大致)不变，长度缩放')
plt.show()

# 验证：2v 的长度确实是 v 的两倍
print(f"‖v‖   = {np.linalg.norm(v):.3f}")
print(f"‖2v‖  = {np.linalg.norm(2*v):.3f}")
print(f"‖-0.5v‖ = {np.linalg.norm(-0.5*v):.3f}")
```

### 4.3　向量减法——向"对面"走

📐 **定义　向量减法（Vector Subtraction）**：`**a** − **b** = **a** + (−**b**)`。几何上，`**a** − **b**` 是从 **b** 的终点指向 **a** 的终点的向量。

💻 **代码　减法与几何意义**

```python
import numpy as np
import matplotlib.pyplot as plt

a = np.array([3, 2])
b = np.array([1, 1])
diff = a - b  # [2, 1]——从 b 终点指向 a 终点的向量

fig, ax = plt.subplots(figsize=(5, 5))
ax.set_xlim(0, 5); ax.set_ylim(0, 4)
ax.quiver(0, 0, a[0], a[1], angles='xy', scale_units='xy', scale=1, color='b', width=0.02, label='a')
ax.quiver(0, 0, b[0], b[1], angles='xy', scale_units='xy', scale=1, color='r', width=0.02, label='b')
# 差向量：从 b 终点出发
ax.quiver(b[0], b[1], diff[0], diff[1], angles='xy', scale_units='xy', scale=1,
          color='green', width=0.02, label='a - b')
ax.legend(); ax.set_aspect('equal')
ax.set_title('a - b：从 b 终点指向 a 终点')
plt.show()
```

### 4.4　AI 应用：词嵌入语义运算

向量运算最惊艳的 AI 应用是**词嵌入（Word Embedding）** 的语义算术。每个词被映射为一个高维向量（如 300 维），向量的方向编码了语义。著名的等式：

> **国王** − **男人** + **女人** ≈ **女王**

本质上就是：从"国王"向量中减去"男性"成分，加上"女性"成分，得到"女王"。

💻 **代码　用 GloVe 预训练词向量验证语义运算**

```python
# 本章使用随机向量模拟语义运算的原理
# 第四节将演示：向量加减法如何编码语义变换
import numpy as np

# 模拟词向量（实际应用中从 GloVe/Word2Vec 加载）
# 简化到 4 维便于展示，实际通常是 100-300 维
king    = np.array([0.5, 0.7, 0.3, 0.2])   # 国王
man     = np.array([0.3, 0.4, 0.1, 0.1])   # 男人
woman   = np.array([0.2, 0.5, 0.2, 0.2])   # 女人

# 语义运算：国王 - 男人 + 女人
queen_pred = king - man + woman
print(f"king:             {king}")
print(f"king - man + woman: {queen_pred}")

# 用余弦相似度找最近的真实词向量（简化：假设已有 queen 向量）
queen_true = np.array([0.4, 0.8, 0.4, 0.3])  # 女王（真实向量）
cos_sim = np.dot(queen_pred, queen_true) / (np.linalg.norm(queen_pred) * np.linalg.norm(queen_true))
print(f"预测向量与真实 queen 的余弦相似度: {cos_sim:.4f}")
# 高相似度 → 向量运算确实编码了语义变换
```

🔗 **AI 连接**：这个等式的实际效果取决于词向量的训练质量。在第 29 章 Transformer 中，Self-Attention 本质上也是在"让每个词去关注其他所有词"，并做类似的向量加权组合——只是权重不是 ±1，而是动态计算的注意力分数。
